### 1. Source Dropdown UI

### 2. Custom Molecule Availability
expected: After uploading and generating a custom molecule in Tab 3 (Custom Molecule tab), the "Insert" button in Tab 4 (Solute tab) should enable when you switch the source to "Custom Molecule". The button should be disabled if no custom molecules are available.
result: ✅ passed (2026-06-24, WF8a — etoh uploaded in Tab 3, Tab 4 Custom Molecule source enabled)

### 3. Source Switching Behavior
expected: When you switch the source dropdown from "Interface" to "Custom Molecule" (or vice versa) in Tab 4, the 3D viewer should clear to prevent showing mixed structures from different sources.
result: ✅ passed (2026-06-24, WF6 — viewer clears on source switch)

### 4. Custom Molecule Source Insertion
expected: After generating a custom molecule in Tab 3 and switching Tab 4 source to "Custom Molecule", clicking "Insert Solute" should insert solutes using the custom molecule structure (not the interface structure). The log message should indicate "Using source: Custom Molecule".

result: ✅ passed (2026-06-24, WF8a — full chain verified: SOL 11570 → etoh 3 → THF_L 36 → NA 6 → CL 6)
