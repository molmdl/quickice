Create `quickice/phase_mapping/ice_ih_density.py` implementing the IAPWS R10-06(2009) Ice Ih density calculation via GSW.

The module must include:

1. **Constants**:
   - `FALLBACK_DENSITY_GCM3 = 0.9167` — g/cm³ at 273.15K, 1 atm (matches old hardcoded value)
   - `DBAR_PER_MPA = 100.0` — unit conversion: 1 MPa = 100 dbar
   - `ATM_PRESSURE_DBAR = 10.1325` — 1 atm in dbar (101325 Pa)
   - `REFERENCE_DENSITIES` dict — verified GSW 3.6.21 reference values for testing:
     ```
     (100, 0.101325): 0.93305,
     (150, 0.101325): 0.93061,
     (200, 0.101325): 0.92613,
     (250, 0.101325): 0.92000,
     (270, 0.101325): 0.91718,
     (273.15, 0.101325): 0.91672,
     (251, 200): 0.93983,
     (273.15, 200): 0.93737,
     ```

2. **`ice_ih_density_kgm3(T_K, P_MPa)`**: Core function, cached with `@lru_cache(maxsize=256)`.
   - Convert T_K → °C: `t_C = T_K - 273.15`
   - Convert P_MPa → sea pressure dbar: `P_sea_dbar = P_MPa * DBAR_PER_MPA - ATM_PRESSURE_DBAR`
   - Call `gsw.rho_ice(t_C, P_sea_dbar)` → density in kg/m³
   - Wrap in try/except catching (ValueError, OverflowError) → return `FALLBACK_DENSITY_GCM3 * 1000`
   - Also catch ImportError (if gsw not installed) → return fallback
   - Return `float(result)`

3. **`ice_ih_density_gcm3(T_K, P_MPa)`**: Convenience function returning g/cm³ (QuickIce standard units).
   - Simply calls `ice_ih_density_kgm3(T_K, P_MPa) / 1000.0`
   - NOT cached separately (the kg/m³ function is cached)

CRITICAL PITFALLS (from research):
- GSW uses °C (ITS-90), NOT Kelvin. Passing K directly will return silently wrong values.
- GSW uses "sea pressure" (absolute - 10.1325 dbar), NOT absolute pressure. At 1 atm (0.101325 MPa), sea pressure = 0 dbar.
- The fallback should be approximately 0.917 g/cm³ (0.9167 specifically, matching the old hardcoded value at 273.15K/1atm).
- DO NOT implement the IAPWS Gibbs equation manually. Use gsw.rho_ice() exclusively.
- DO NOT call gsw.rho_ice() directly from outside this module. Always go through the wrapper.
