**Color constants:**

```python
ICE_COLOR = (0.0, 0.8, 0.8)         # Cyan
WATER_COLOR = (0.39, 0.58, 0.93)    # Cornflower blue
BOND_LINE_WIDTH = 1.5
ANGSTROM_TO_NM = 0.1
```

**Class InterfaceViewerWidget(QWidget):**
- `__init__(self, parent=None)`: Call super, init state vars, call _setup_vtk()
- State vars: _current_structure (InterfaceStructure|None), _ice_actor, _water_actor, _ice_bond_actor, _water_bond_actor, _unit_cell_actor (all vtkActor|None, init None)

**`_setup_vtk(self)`:** Follow same pattern as MolecularViewerWidget._setup_vtk():
- Create QVTKRenderWindowInteractor as self.vtk_widget
- QVBoxLayout(self), add widget, setContentsMargins(0,0,0,0)
- Get render_window, create vtkRenderer with background (0.1, 0.2, 0.4) — same as Tab 1
- Get interactor, set vtkInteractorStyleTrackballCamera
- Call self.vtk_widget.Initialize()

**`set_interface_structure(self, structure: InterfaceStructure)`:**
- Call _clear_actors()
- Store self._current_structure = structure
- Call interface_to_vtk_molecules(structure) → ice_mol, water_mol
- Create ice actor: _create_phase_actor(ice_mol, ICE_COLOR) → self._ice_actor
- Create water actor: _create_phase_actor(water_mol, WATER_COLOR) → self._water_actor
- Extract bonds from each molecule: _extract_bonds(mol) → bond_pairs list
- Create bond actors: create_bond_lines_actor(ice_bonds, ICE_COLOR, BOND_LINE_WIDTH) → self._ice_bond_actor
- Same for water_bonds → self._water_bond_actor
- Create unit cell actor: create_unit_cell_actor(structure.cell) → self._unit_cell_actor
- Add all 5 actors to renderer
- Call _set_side_view_camera()
- Call self.render_window.Render()

**`_create_phase_actor(self, mol, color_rgb) -> vtkActor`:**
- Create vtkMoleculeMapper, SetInputData(mol)
- SetAtomColorModeToSingleColor()
- SetAtomColor(r, g, b) as uint8 (multiply each float by 255, cast to int)
- RenderBondsOff() — bonds rendered separately as lines
- SetAtomicRadiusTypeToUnitRadius()
- SetAtomicRadiusScaleFactor(0.2 * ANGSTROM_TO_NM) — small spheres for performance
- Create vtkActor, SetMapper, return

**`_extract_bonds(self, mol) -> list`:**
- Iterate mol.GetNumberOfBonds(), for each bond get BeginAtomId/EndAtomId
- Get positions via mol.GetAtom(id).GetPosition()
- Return list of ((x1,y1,z1), (x2,y2,z2)) tuples

**`_set_side_view_camera(self)`:**
- Call self.renderer.ResetCamera() FIRST (to compute bounds/center)
- Get bounds via self.renderer.ComputeVisiblePropBounds()
- Calculate center_x, center_y, center_z from bounds
- Calculate diagonal from bounds (sqrt of sum of squared dimensions)
- Get camera, set:
  - Position: (center_x, center_y + diagonal, center_z)
  - FocalPoint: (center_x, center_y, center_z)
  - ViewUp: (0, 0, 1) — Z is vertical (slab stacking direction)
- Call self.renderer.ResetCameraClippingRange() (for proper near/far planes)

**`_clear_actors(self)`:**
- Remove all 5 actors from renderer if not None (ice, water, ice_bonds, water_bond, unit_cell)
- Set all to None
- Set self._current_structure = None

**`clear(self)`:**
- Call _clear_actors()
- Call self.render_window.Render()

**`reset_camera(self)`:**
- Call _set_side_view_camera()
- Call self.render_window.Render()

**Imports needed:**
- PySide6.QtWidgets: QWidget, QVBoxLayout
- vtkmodules.qt.QVTKRenderWindowInteractor: QVTKRenderWindowInteractor
- vtkmodules.all: vtkRenderer, vtkInteractorStyleTrackballCamera, vtkMoleculeMapper, vtkActor
- quickice.structure_generation.types: InterfaceStructure
- quickice.gui.vtk_utils: interface_to_vtk_molecules, create_bond_lines_actor, create_unit_cell_actor
