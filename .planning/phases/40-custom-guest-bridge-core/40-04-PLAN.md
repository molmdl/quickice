<objective>
Create `custom_guest_bridge.py` — the core module that (1) builds a synthetic GenIce2 `Molecule` plugin module from a GRO file (centered), (2) validates a GRO+ITP pair (parseable, atom count, name length <=3 chars, comb-rule=2), and (3) injects/cleans up the module in `sys.modules` thread-safely.

Purpose: This is the heart of Phase 40. GenIce2's `safe_import('molecule', <guest_type>)` checks `sys.modules` first — by injecting a synthetic `types.ModuleType` with a `Molecule` subclass, GenIce2 places the custom guest in cage positions during `generate_ice()`. The mechanism was verified end-to-end in research (POC generated a real sI hydrate with 2 ethanol molecules in cages). The bridge must register on the main thread (thread-safe per v4.7 decision) and clean up after generation (no stale module pollution).

Output: `custom_guest_bridge.py` with build/validate/inject/cleanup primitives + unit tests covering all validation reject paths and sys.modules lifecycle.
</objective>

3. Center the molecule (MANDATORY — GenIce2's `arrange()` adds cage center to sites_; sites_ must be relative to centroid): `centroid = positions.mean(axis=0)` then `centered = positions - centroid`.
4. Validate guest_type name: `from genice2.plugin import audit_name` (lazy import). Assert `audit_name(guest_type)` — if False, raise `ValueError(f"Invalid guest type name '{guest_type}'; allowed chars: A-Z a-z 0-9 _ -")`. audit_name requires `^[A-Za-z0-9-_]+$`.
5. Build the synthetic module: `mod_key = f"genice2.molecules.{guest_type}"`, `mod = types.ModuleType(mod_key)`. Use `exec` to define a `Molecule` subclass in `mod.__dict__` (per research POC):
   ```python
   code = (
       "import numpy as np\nimport genice2.molecules\n"
       f"class Molecule(genice2.molecules.Molecule):\n"
       f"    '''Custom guest {residue_name} (generated by QuickIce).'''\n"
       "    def __init__(self):\n"
       f"        self.sites_ = np.array({centered.tolist()!r})\n"
       f"        self.labels_ = {list(atom_names)!r}\n"
       f"        self.name_ = {residue_name!r}\n"
   )
   exec(code, mod.__dict__)
   ```
6. Return `mod` (the caller registers it in sys.modules — this function does NOT register, only builds).

Do NOT auto-convert Å→nm (GRO spec is nm; parse_gro_file already rejects >50nm). Do NOT mutate the input positions (parse_gro_file returns a fresh array).

Add three primitives to `custom_guest_bridge.py`:

1. `custom_guest_module(guest_type, module)` — a context manager (use `from contextlib import contextmanager`):
   ```python
   @contextmanager
   def custom_guest_module(guest_type, module):
       key = f"genice2.molecules.{guest_type}"
       assert key not in sys.modules, f"{key} already registered (stale state?)"
       sys.modules[key] = module
       try:
           yield key
       finally:
           sys.modules.pop(key, None)
   ```

2. `register_custom_guest(guest_type, module)` — for GUI async (QThread): `sys.modules[f"genice2.molecules.{guest_type}"] = module`. Call BEFORE thread.start().

3. `unregister_custom_guest(guest_type)` — for GUI QThread finished signal: `sys.modules.pop(f"genice2.molecules.{guest_type}", None)`. Safe if absent.
