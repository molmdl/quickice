2. Wrap the `_run_via_api` call in the `custom_guest_module` context manager when `config.is_custom_guest` is True. The current code is:
   ```python
   positions, cell, atom_names, residue_names, residue_seq_nums = self._run_via_api(lattice_name, config)
   ```
   Change to:
   ```python
   if config.is_custom_guest:
       # Build the synthetic Molecule module on the MAIN thread (thread-safe per v4.7).
       # _run_via_api already passes guest_type through to parse_guest (else branch).
       module = build_custom_guest_module(
           config.guest_gro_path, config.guest_type, config.guest_residue_name
       )
       with custom_guest_module(config.guest_type, module):
           positions, cell, atom_names, residue_names, residue_seq_nums = self._run_via_api(lattice_name, config)
   else:
       positions, cell, atom_names, residue_names, residue_seq_nums = self._run_via_api(lattice_name, config)
   ```
   The context manager handles sys.modules registration BEFORE generate_ice runs and cleanup (try/finally) AFTER. _build_molecule_index and HydrateStructure construction happen AFTER the context manager exits (they don't need the module — only generate_ice does).

   IMPORTANT: Do NOT change `_run_via_api` itself — it already passes `guest_type` through to `parse_guest` via the else branch (`guest_name = guest_type`). The only requirement is that `sys.modules["genice2.molecules.<guest_type>"]` is populated before `ice.generate_ice(...)` runs, which the context manager guarantees.

   ```python
   lines.extend([
   ```

   ```python
   if config.is_custom_guest:

           f"  Small cage occupancy: {config.cage_occupancy_small}%",
           f"  Large cage occupancy: {config.cage_occupancy_large}%",
       ])
   ```

Use `except (ValueError, OSError, KeyError)` not bare `except` anywhere in the added code (AGENTS.md). The existing `except Exception` in `_run_via_api` is pre-existing GUI-adjacent code — do NOT touch it in this plan (it's wrapping GenIce2 calls for user-facing error messages; leave as-is to minimize regression risk).
