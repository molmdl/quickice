Create output_ranked_candidates function that coordinates all output operations:

```python
def output_ranked_candidates(
    ranking_result: RankingResult,
    output_dir: str,
    base_name: str = "ice_candidate",
    generate_diagram: bool = True,
    user_t: float | None = None,
    user_p: float | None = None,
) -> OutputResult:
```

**Implementation flow:**
1. Create output directory with Path(output_dir).mkdir(parents=True, exist_ok=True)
2. Initialize empty lists: output_files, phase_diagram_files, validation_results
3. For each ranked_candidate in ranking_result.ranked_candidates[:10]:
   - Get candidate = ranked_candidate.candidate, rank = ranked_candidate.rank
   - Write PDB: filename = f"{base_name}_{rank:02d}.pdb"
   - Call write_pdb_with_cryst1(candidate, filepath)
   - Validate: val_result = validate_space_group(candidate)
   - Check overlap: has_overlap = check_atomic_overlap(candidate)
   - Store validation: validation_results.append({'rank': rank, 'file': filename, 'spacegroup': val_result['spacegroup_symbol'], 'valid': val_result['valid'] and not has_overlap, 'has_overlap': has_overlap})
   - Append filepath to output_files
4. If generate_diagram and user_t and user_p:
   - Call generate_phase_diagram(user_t, user_p, output_dir)
   - phase_diagram_files = [phase_diagram.png, phase_diagram.svg, phase_diagram_data.txt]
5. Create summary: {'n_candidates': len(output_files), 'n_validated': sum(1 for v in validation_results if v['valid']), 'user_T': user_t, 'user_P': user_p}
6. Return OutputResult(output_files, phase_diagram_files, validation_results, summary)

**Error handling:**
- Warn on validation failures (don't fail)
- Use logging.warning() for issues
- Continue on individual file errors
