<feature>
  <name>Structure Validator</name>
  <files>quickice/output/validator.py, tests/test_output/test_validator.py</files>
  <behavior>
    Validate ice structures for crystal symmetry and atomic overlaps.
    
    Cases:
    1. validate_space_group(candidate) -> dict with spacegroup_number, spacegroup_symbol, valid
    2. check_atomic_overlap(candidate, min_distance=0.8) -> bool (True if overlap detected)
    3. Uses symprec=1e-4 for generated structures (not too strict)
    4. Minimum image convention for PBC distance calculations
    
    Critical: Cell and positions must be converted to Angstrom for spglib (multiply by 10.0).
    Critical: min_distance=0.8 Å to avoid false positives from O-H bonds (~0.96 Å).
  </behavior>
  <implementation>
Implementation steps after tests pass:

1. **validate_space_group(candidate, symprec=1e-4)**:
   - Import spglib
   - Convert cell and positions from nm to Angstrom: lattice = candidate.cell * 10.0, positions = candidate.positions * 10.0
   - Map atom names to atomic numbers: {'O': 8, 'H': 1}
   - Create atom_types array
   - Call spglib.get_dataset(lattice, positions, atom_types, symprec=symprec)
   - If dataset is None, return {'spacegroup_number': None, 'spacegroup_symbol': 'UNKNOWN', 'valid': False}
   - Else return {'spacegroup_number': dataset.spacegroup_number, 'spacegroup_symbol': dataset.spacegroup_symbol, 'hall_number': dataset.hall_number, 'n_operations': len(dataset.rotations), 'valid': True}

2. **check_atomic_overlap(candidate, min_distance=0.8)**:
   - Convert to Angstrom: positions = candidate.positions * 10.0, cell = candidate.cell * 10.0
   - Calculate inverse cell: inv_cell = np.linalg.inv(cell)
   - Convert to fractional coordinates: frac = positions @ inv_cell
   - For each pair (i, j):
     - Calculate delta = frac[j] - frac[i]
     - Apply minimum image convention: delta = delta - np.floor(delta + 0.5)
     - Convert back: cart_delta = delta @ cell
     - Calculate distance: dist = np.linalg.norm(cart_delta)
     - If dist < min_distance, return True (overlap detected)
   - Return False (no overlap)

3. **Edge Cases**:
   - Empty positions list: return False (no overlap possible)
   - spglib returns None: validation fails but continues
   - Single atom: no overlap check needed
</implementation>
</feature>
