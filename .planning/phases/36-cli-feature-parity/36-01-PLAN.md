Add 6 new argument groups to `create_parser()` and create `validate_pipeline_args()` function.

**New argument groups (add AFTER the existing interface_group, BEFORE --version):**

1. **hydrate generation** group:
   - `--hydrate` action="store_true", help="Generate hydrate structure"
   - `--lattice-type` choices=["sI", "sII", "sH"], default="sI", help="Hydrate lattice type"
   - `--guest` choices=["CH4", "THF"], default="CH4", help="Guest molecule type"
   - `--supercell-x`, `--supercell-y`, `--supercell-z` type=int, default=1, help="Supercell dimensions"
   - `--cage-occupancy-small` type=float, default=100.0, help="Small cage occupancy 0-100%"
   - `--cage-occupancy-large` type=float, default=100.0, help="Large cage occupancy 0-100%"

2. **custom molecule insertion** group:
   - `--custom-gro` type=str, help="Path to custom molecule .gro file"
   - `--custom-itp` type=str, help="Path to custom molecule .itp file"
   - `--custom-placement` choices=["random", "custom"], default="random", help="Placement mode"
   - `--custom-count` type=int, help="Molecule count for random mode"
   - `--custom-concentration` type=float, help="Concentration in mol/L (random mode, mutually exclusive with --custom-count)"
   - `--custom-positions-file` type=str, help="CSV file with x,y,z,alpha,beta,gamma columns (custom mode)"

3. **solute insertion** group:
   - `--solute-type` choices=["CH4", "THF"], help="Solute molecule type"
   - `--solute-concentration` type=float, help="Solute concentration in mol/L"
   - `--solute-source` choices=["interface", "custom"], default="interface", help="Source structure for solute insertion"

4. **ion insertion** group:
   - `--ion-concentration` type=float, help="NaCl concentration in mol/L"
   - `--ion-source` choices=["interface", "custom", "solute"], default="interface", help="Source structure for ion insertion"

5. **output options** group (extend existing, add --no-overwrite):
   - `--no-overwrite` action="store_true", default=False, help="Error if output files exist (default: auto-overwrite)"

**Update `validate_interface_args()` → rename to `validate_pipeline_args()`:**

Keep ALL existing validation rules. Add these new rules (errors use `parser.error()` which exits with code 2):

- `--hydrate` + `--nmolecules` is invalid (hydrate uses supercell, not molecule count)
- `--hydrate` requires `--lattice-type` is always set (has default, so just validate choices are valid)
- `--hydrate` + `--interface` is valid (hydrate→interface pipeline)
- `--custom-gro` requires `--custom-itp` and vice versa (both or neither)
- `--custom-gro` requires `--interface` (custom molecules go into interface structures)
- `--custom-placement custom` requires `--custom-positions-file`
- `--custom-placement random` requires either `--custom-count` or `--custom-concentration` (but not both)
- `--custom-count` and `--custom-concentration` are mutually exclusive
- `--solute-type` requires `--solute-concentration`
- `--solute-type` requires `--interface` (solutes go into interface structures)
- `--solute-source custom` requires `--custom-gro` (custom step must have run)
- `--ion-concentration` requires `--interface`
- `--ion-source custom` requires `--custom-gro`
- `--ion-source solute` requires `--solute-type`
