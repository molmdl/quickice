  <action>
Refactor `quickice/main.py` to delegate to `CLIPipeline` while preserving backward compatibility with existing ice-only and interface-only usage.

**Changes to main():**

1. Replace the monolithic interface generation code (lines 73-160) with CLIPipeline delegation. Add import at top:
   ```python
   from quickice.cli.pipeline import CLIPipeline
   ```

2. Add a check: if any v4.5 flags are present (--interface, --hydrate, --custom-gro, --solute-type, --ion-concentration), delegate to CLIPipeline:
   ```python
   # Detect if any pipeline flags are present (v4.5 multi-step workflow)
   has_pipeline_flags = (
       args.interface
       or getattr(args, 'hydrate', False)
       or getattr(args, 'custom_gro', None) is not None
       or getattr(args, 'solute_type', None) is not None
       or getattr(args, 'ion_concentration', None) is not None
   )
   
   if has_pipeline_flags:
       pipeline = CLIPipeline(args)
       return pipeline.execute()
   ```

3. Keep the existing ice-only path (lines 162-271) for backward compatibility when only `--temperature`, `--pressure`, `--nmolecules` are specified (no new flags). This preserves the existing candidate generation + ranking + GROMACS export + PDB output behavior.

4. Remove the `check_output_file()` function entirely (replace with auto-overwrite behavior per CONTEXT.md). The CLIPipeline handles `--no-overwrite` internally.

5. Update the `main()` try/except structure:
   - `SystemExit` from argparse: re-raise (exit code 2 from parser.error)
   - `UnknownPhaseError`: print to stderr, return 1
   - `InterfaceGenerationError`: print to stderr, return 1
   - Generic `Exception`: print to stderr, return 1

6. The `main()` function should still print initial phase info (temperature, pressure, phase lookup) before delegating to CLIPipeline, so users see phase identification on stdout. BUT ã move the phase lookup INSIDE CLIPipeline.execute() since it needs phase_info for ice candidate generation. Keep the temperature/pressure print in main() for the ice-only path.

**Critical constraint:** NO imports from `quickice/gui/` in main.py. Only import from `quickice.cli.pipeline`, `quickice.cli.parser`, `quickice.phase_mapping`, `quickice.structure_generation`, `quickice.output.gromacs_writer`.
