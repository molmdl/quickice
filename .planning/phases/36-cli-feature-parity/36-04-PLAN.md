Add to existing `itp_helpers.py` (the 3 path resolvers from Plan 02 are already there).

**Private helper `_detect_guest_type(structure) -> str | None`:**
- If structure lacks `guest_atom_count` AND `molecule_index`, delegate to `structure.interface_structure` (SoluteStructure fallback — FIX #5)
- If `molecule_index` exists and is non-empty: iterate entries, use `entry.mol_type.upper()` (**NOT** `entry.get('mol_type')` — MoleculeIndex is a dataclass, FIX #2). Check for 'CH4' → 'ch4', 'THF' → 'thf'
- Fallback: atom count heuristic using `getattr(structure, 'guest_nmolecules', 1)` (**NOT** bare `structure.guest_nmolecules` — FIX #8). `atoms_per = guest_atom_count / max(guest_nmolecules, 1)`. 5 atoms → 'ch4', 13 → 'thf'
- Return None if no guest detected

**`copy_itp_files_for_structure(output_dir: Path, structure, step_name: str) -> list[str]`:**
Returns list of ITP filenames copied. Cases:

1. **"ice"**: Copy `tip4p-ice.itp` only

2. **"hydrate"** (NEW — FIX #10, was MISSING):
   - Copy `tip4p-ice.itp`
   - Copy hydrate guest ITP via `get_hydrate_guest_itp_path()`. Detect guest type: if structure has `guest_type` attr → use it (call .lower()); else try `_detect_guest_type(structure)`; else use `getattr(args_ref, 'guest', 'CH4').lower()` as last resort. For HydrateStructure, `guest_type` attribute exists directly.

3. **"interface"**: Copy `tip4p-ice.itp` + if guests present (`guest_atom_count > 0`), copy hydrate guest ITP via `_detect_guest_type(structure)`

4. **"custom"**: Copy `tip4p-ice.itp` + custom ITP (from `structure.itp_path` with `comment_out_atomtypes_in_itp()`) + hydrate guest ITP if guests present

5. **"solute"**: Copy `tip4p-ice.itp` + solute liquid ITP (via `get_solute_liquid_itp_path()`, with atomtypes commented) + custom ITP if `getattr(structure, 'custom_molecule_count', 0) > 0` (from `structure.custom_itp_path`, atomtypes commented) + hydrate guest ITP if guests present

6. **"ion"**: Copy `tip4p-ice.itp` + generate `ion.itp` via `write_ion_itp(str(output_dir / "ion.itp"), structure.na_count, structure.cl_count)` + all solute/custom/hydrate guest ITPs (same logic as solute case)

**Import:** `from quickice.output.gromacs_writer import comment_out_atomtypes_in_itp` and `from quickice.structure_generation.gromacs_ion_export import write_ion_itp`

**Error handling:** If ITP file not found, `logger.warning(...)` but don't fail — return list of successfully copied filenames.
