**`execute() -> int`** — runs steps in order, fail-fast:
1. Create output dir: `Path(self.args.output).resolve().mkdir(parents=True, exist_ok=True)`
2. Check `--no-overwrite` if `getattr(self.args, 'no_overwrite', False)`
3. Step 1: if `args.interface or getattr(args, 'hydrate', False)` → `_run_source_step()`, return on error
4. Step 2: if `args.interface` → `_run_interface_step()`, return on error
5. Step 3: if `getattr(args, 'custom_gro', None)` → `_run_custom_step()`, return on error
6. Step 4: if `getattr(args, 'solute_type', None)` → `_run_solute_step()`, return on error
7. Step 5: if `getattr(args, 'ion_concentration', None)` → `_run_ion_step()`, return on error
8. Step 6: `_run_export_step()`, return result

**`_get_source_structure(source_name: str)`** — returns stored structure:
- "interface" → `self._interface_result`
- "custom" → `self._custom_result`
- "solute" → `self._solute_result`
- Raise ValueError for unknown names

**`_parse_positions_csv(filepath) -> tuple[list[tuple], list[tuple]]`** — static method:

- Raise FileNotFoundError if file doesn't exist
