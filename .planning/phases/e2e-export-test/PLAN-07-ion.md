**Key differences from other exporters:**
1. `ion.itp` is GENERATED via `write_ion_itp()`, not copied from data directory
2. All conditional ITP paths (guest, solute, custom) are present
3. The exporter checks `ion_structure.molecule_index` for mol_type="guest" entries (lines 346-352)
4. The solute ITP source is found via `Path(__file__).parent.parent / "data" / f"{solute_type_lower}_liquid.itp"` (line 381)

**Guest detection in IonGROMACSExporter (lines 338-373):**
- First checks `ion_structure.guest_nmolecules > 0 AND ion_structure.guest_atom_count > 0`
- Then iterates `ion_structure.molecule_index` for `mol_type == "guest"` entries
- Extracts atom names for each guest molecule and calls `detect_guest_type_from_atoms()`
- Fallback heuristic: `atoms_per_guest >= 10 → "thf", else "ch4"`
