    4. is_triclinic() returns False for near-orthogonal (89.95°, 90.05°, 90.0°)

    7. transform_if_needed() produces orthogonal cell for Ice V

    9. get_cell_angles() computes correct angles
    10. get_cell_extent() computes bounding box correctly
  </behavior>
  <implementation>
    Follow the code patterns from RESEARCH.md:
    - Use numpy for all matrix operations
    - ANGLE_TOL_DEG = 0.1 for orthogonal detection
    - Calculate angles using np.clip for numerical stability
    - Use integer supercell matrices for transformation
    - Implement validation comparing before/after density and structure

    Architecture:
    ```
    transformer_types.py:
      - TransformationStatus enum (SKIPPED, TRANSFORMED, FAILED)
      - TransformationResult dataclass (status, cell, positions, multiplier, message)

    transformer.py:
      - TriclinicTransformer class
        - ANGLE_TOL_DEG = 0.1
        - is_triclinic(cell) -> bool
        - get_cell_angles(cell) -> (alpha, beta, gamma)
        - get_cell_extent(cell) -> (dx, dy, dz)
        - find_orthogonal_supercell(cell, max_multiplier=50) -> H | None
        - apply_transformation(positions, cell, H) -> (new_positions, new_cell)
        - validate_transformation(orig, transformed) -> bool
        - transform_if_needed(candidate) -> TransformationResult
    ```

    For transformation matrix discovery, start with known patterns:
    - Ice II (rhombohedral): Try rhombohedral→hexagonal→orthogonal chain
    - Ice V (monoclinic): Try integer combinations to cancel tilt

class TestCellAngleCalculation:
    """Tests for cell angle computation."""

    def test_orthogonal_cell_has_90_degree_angles(self):
        """Orthogonal cell should have all angles at 90°."""
        cell = np.diag([1.0, 2.0, 3.0])  # Perfect orthogonal
        transformer = TriclinicTransformer()
        alpha, beta, gamma = transformer.get_cell_angles(cell)
        assert alpha == pytest.approx(90.0, abs=0.01)
        assert beta == pytest.approx(90.0, abs=0.01)
        assert gamma == pytest.approx(90.0, abs=0.01)

    def test_rhombohedral_cell_has_equal_angles(self):
        """Ice II rhombohedral cell should have α=β=γ≈113°."""
        # Create rhombohedral cell with α=β=γ=113°
        angle_rad = np.radians(113.0)
        a = 1.0
        cell = np.array([
            [a, 0, 0],
            [a * np.cos(angle_rad), a * np.sin(angle_rad), 0],
            [a * np.cos(angle_rad), 
             a * (np.cos(angle_rad) - np.cos(angle_rad)**2) / np.sin(angle_rad),
             a * np.sqrt(1 - 2*np.cos(angle_rad)**2 + np.cos(angle_rad)**4) / np.sin(angle_rad)]
        ])
        transformer = TriclinicTransformer()
        alpha, beta, gamma = transformer.get_cell_angles(cell)
        assert alpha == pytest.approx(113.0, abs=0.5)
        assert beta == pytest.approx(113.0, abs=0.5)
        assert gamma == pytest.approx(113.0, abs=0.5)
