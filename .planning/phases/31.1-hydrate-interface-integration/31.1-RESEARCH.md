### Pattern 1: Stacked Placeholder/3D Viewer

**What:** QStackedWidget with index 0 = placeholder (text message), index 1 = VTK 3D viewer

**When to use:** Any tab that generates structures - shows placeholder before generation, 3D viewer after

**Example (from HydrateViewerWidget):**
```python
from PySide6.QtWidgets import QWidget, QVBoxLayout, QStackedWidget, QLabel
from PySide6.QtCore import Qt

class HydrateViewerWidget(QWidget):
    def __init__(self, parent=None):
        super().__init__(parent)
        self._setup_ui()
    
    def _setup_ui(self):
        main_layout = QVBoxLayout(self)
        main_layout.setContentsMargins(0, 0, 0, 0)
        
        self._stack = QStackedWidget()
        main_layout.addWidget(self._stack)
        
        # Index 0: Placeholder
        placeholder = self._create_placeholder_widget()
        self._stack.addWidget(placeholder)
        
        # Index 1: VTK viewer
        if self._vtk_available:
            viewer = self._create_vtk_viewer_widget()
        else:
            viewer = self._create_vtk_fallback_widget()
        self._stack.addWidget(viewer)
        
        # Show placeholder initially
        self._stack.setCurrentIndex(0)
```

### Pattern 2: Ion Rendering

**What:** Use existing `render_ion_structure()` from ion_renderer.py to create VTK actors

**When to use:** Displaying Na+/Cl- ions in 3D viewer after insertion

**Existing code (ion_renderer.py lines 177-225):**
```python
from quickice.gui.ion_renderer import render_ion_structure

def render_ion_structure(ion_structure) -> list[vtkActor]:
    """Render Na+ and Cl- ions as VDW spheres from an IonStructure."""
    actors = []
    na_positions = []
    cl_positions = []
    
    for i, name in enumerate(ion_structure.atom_names):
        if name == "NA":
            na_positions.append(ion_structure.positions[i])
        elif name == "CL":
            cl_positions.append(ion_structure.positions[i])
    
    # Create Na+ actors (gold VDW spheres)
    for pos in na_positions:
        actor = _create_vdw_sphere(
            center=(float(pos[0]), float(pos[1]), float(pos[2])),
            radius=NA_VDW_RADIUS,  # 0.190 nm
            color_rgb=NA_COLOR,    # (1.0, 0.84, 0.0) gold
        )
        actors.append(actor)
    
    # Create Cl- actors (green VDW spheres)
    for pos in cl_positions:
        actor = _create_vdw_sphere(
            center=(float(pos[0]), float(pos[1]), float(pos[2])),
            radius=CL_VDW_RADIUS,  # 0.181 nm
            color_rgb=CL_COLOR,    # (0.56, 0.99, 0.56) lime green
        )
        actors.append(actor)
    
    return actors
```

## Common Pitfalls

### Pitfall 1: VTK Availability Check Missing
**What goes wrong:** Viewer crashes in headless/SSH environments
**Why it happens:** No VTK availability check before creating QVTKRenderWindowInteractor
**How to avoid:** Follow HydrateViewerWidget's pattern - check _VTK_AVAILABLE flag, create fallback widget if unavailable
**Warning signs:** `qt.qpa.xcb: cannot connect to display` errors

### Pitfall 2: Ion Structure Not Passed to Viewer
**What goes wrong:** Viewer exists but shows nothing after ion insertion
**Why it happens:** render_ion_structure() result not passed to viewer widget
**How to avoid:** Call viewer.set_ion_structure(ion_structure) after insertion completes
**Warning signs:** No error, but empty viewer after "Insert Ions" click

### Pitfall 3: Wrong Viewer Used for Ion Display
**What goes wrong:** Ions render in Tab 1 viewer instead of IonPanel viewer
**Why it happens:** Current code (main_window.py lines 700-718) adds actors to Tab 1's dual_viewer instead of IonPanel's viewer
**How to avoid:** Add dedicated viewer to IonPanel, render ions there after insertion

### Pitfall 4: Camera Not Reset After Structure Load
**What goes wrong:** Structure appears off-screen or too small/large
**Why it happens:** No camera auto-fit after loading new structure
**How to avoid:** Call _reset_camera() after adding actors to renderer (matches HydrateViewerWidget pattern)

### Creating IonViewerWidget (recommended pattern)

```python
# File: quickice/gui/ion_viewer.py
# Following hydrate_viewer.py pattern

from PySide6.QtWidgets import QWidget, QVBoxLayout, QStackedWidget, QLabel
from PySide6.QtCore import Qt

# VTK availability check (same as other viewers)
import os
_VTK_AVAILABLE = False
try:
    if os.environ.get('QUICKICE_FORCE_VTK', '').lower() == 'true':
        _VTK_AVAILABLE = True
    elif os.environ.get('DISPLAY'):
        _VTK_AVAILABLE = True
    
    if _VTK_AVAILABLE:
        from vtkmodules.qt.QVTKRenderWindowInteractor import QVTKRenderWindowInteractor
        from vtkmodules.all import vtkRenderer, vtkInteractorStyleTrackballCamera
except Exception:
    _VTK_AVAILABLE = False

from quickice.structure_generation.types import IonStructure
from quickice.gui.ion_renderer import render_ion_structure


class IonViewerWidget(QWidget):
    """VTK-based 3D viewer for ion structures."""
    
    def __init__(self, parent=None):
        super().__init__(parent)
        self._vtk_available = _VTK_AVAILABLE
        self._current_structure: IonStructure | None = None
        self._ion_actors: list = []
        self._setup_ui()
    
    def _setup_ui(self):
        layout = QVBoxLayout(self)
        layout.setContentsMargins(0, 0, 0, 0)
        
        self._stack = QStackedWidget()
        layout.addWidget(self._stack)
        
        # Index 0: Placeholder
        placeholder = QWidget()
        pl_layout = QVBoxLayout(placeholder)
        pl_layout.setAlignment(Qt.AlignCenter)
        pl_label = QLabel("No ions inserted")
        pl_label.setStyleSheet("color: gray; font-size: 14px;")
        pl_layout.addWidget(pl_label)
        self._stack.addWidget(placeholder)
        
        # Index 1: VTK viewer
        if self._vtk_available:
            viewer = self._create_vtk_viewer_widget()
        else:
            viewer = self._create_vtk_fallback_widget()
        self._stack.addWidget(viewer)
        
        self._stack.setCurrentIndex(0)
    
    def _create_vtk_viewer_widget(self) -> QWidget:
        widget = QWidget()
        self.vtk_widget = QVTKRenderWindowInteractor(widget)
        
        layout = QVBoxLayout(widget)
        layout.addWidget(self.vtk_widget)
        layout.setContentsMargins(0, 0, 0, 0)
        
        self.render_window = self.vtk_widget.GetRenderWindow()
        self.renderer = vtkRenderer()
        self.render_window.AddRenderer(self.renderer)
        self.renderer.SetBackground(0.1, 0.2, 0.4)
        
        self.interactor = self.render_window.GetInteractor()
        style = vtkInteractorStyleTrackballCamera()
        self.interactor.SetInteractorStyle(style)
        self.vtk_widget.Initialize()
        
        return widget
    
    def _create_vtk_fallback_widget(self) -> QWidget:
        widget = QWidget()
        layout = QVBoxLayout(widget)
        layout.setAlignment(Qt.AlignCenter)
        label = QLabel("3D viewer unavailable")
        label.setStyleSheet("color: gray; font-size: 14px;")
        layout.addWidget(label)
        return widget
    
    def set_ion_structure(self, structure: IonStructure) -> None:
        """Display ion structure in viewer."""
        if not self._vtk_available:
            self._current_structure = structure
            self._stack.setCurrentIndex(1)
            return
        
        # Clear previous actors
        for actor in self._ion_actors:
            self.renderer.RemoveActor(actor)
        self._ion_actors = []
        
        self._current_structure = structure
        
        # Render ions
        self._ion_actors = render_ion_structure(structure)
        for actor in self._ion_actors:
            self.renderer.AddActor(actor)
        
        # Reset camera
        self.renderer.ResetCamera()
        
        # Show viewer
        self._stack.setCurrentIndex(1)
        self.render_window.Render()
    
    def clear(self):
        """Clear viewer."""
        for actor in self._ion_actors:
            self.renderer.RemoveActor(actor)
        self._ion_actors = []
        self._current_structure = None
        self._stack.setCurrentIndex(0)
        if self._vtk_available and self.render_window:
            self.render_window.Render()
    
    def is_vtk_available(self) -> bool:
        return self._vtk_available
```

## Open Questions

1. **Hydrate→Interface wiring**
   - What we know: HydrateStructure and InterfaceStructure are different types (hydrate = water framework + guests, interface = ice + water)
   - What's unclear: The original description says "hydrate generated in Tab 2 cannot feed to Interface tab" - but these are fundamentally different structures. Is there a use case for converting hydrate to interface?
   - Recommendation: Treat as separate workflows. If conversion is needed, create separate converter function. Current focus should be ion viewer.

2. **Toolbar for IonPanel viewer**
   - What we know: HydratePanel has H-bonds, unit cell, representation toggles
   - What's unclear: What toggles make sense for ions? (ions are just spheres, no bonds, no H-bonds)
   - Recommendation: Start with minimal toolbar (maybe "reset camera" button only), expand if needed

3. **Ion viewer vs. interface viewer integration**
   - What we know: IonPanel operates on InterfaceStructure (from Tab 3)
   - What's unclear: Should IonPanel viewer show the entire interface (ice + water + ions) or just ions?
   - Recommendation: Show ice + water + ions for context, use distinct colors (gold for Na+, green for Cl-, cyan for ice, blue for water)
