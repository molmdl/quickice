```
## Solute Insertion (Tab 3)

![Solute Panel](images/solute-panel.png)
*Solute insertion panel with concentration input and molecule preview*

### Overview

The Solute Insertion tab allows you to insert THF or CH₄ solutes into liquid water regions at a specified concentration. This is useful for simulating solute effects on ice-water interfaces or hydrate formation.

### Prerequisites

- Interface structure generated in Tab 2 (Interface Construction)
- Liquid water region present in the interface

### Workflow

1. **Generate Interface First**: Create an ice-water interface in Tab 2
2. **Set Concentration**: Enter concentration in mol/L (M)
   - Seawater: ~0.6 M
   - Typical lab solutions: 0.1–2.0 M
3. **Select Solute Type**: Choose THF or CH₄ from dropdown
4. **Preview Molecule Count**: See calculated molecule count in real-time
5. **Insert Solutes**: Click "Insert Solutes" button
6. **Visualize**: View solutes rendered in distinct colors in 3D viewer
7. **Export**: Use Ctrl+L or File menu to export for GROMACS

### Concentration Calculation

Molecule count is calculated using:

```
N = concentration (mol/L) × volume (nm³) × 10⁻²⁴ × N_A
```

where N_A = 6.022×10²³ (Avogadro's number)

Example: 0.6 M in 10 nm³ → ~36 molecules

### Visualization

- THF: Ball-and-stick rendering (gray C, red O, white H)
- CH₄: Ball-and-stick rendering (gray C, white H)
- Both solutes rendered distinctly from water molecules

### GROMACS Export

Exported files include:
- .gro file with ice + water + solutes
- .top file with molecule ordering (SOL → solutes)
- tip4p-ice.itp (water topology)
- ch4.itp or thf.itp (solute topology, GAFF2 parameters)
```

3. Add new section "Custom Molecule Upload (Tab 4)" after Solute Insertion section:
```
## Custom Molecule Upload (Tab 4)

![Custom Molecule Panel](images/custom-molecule-panel.png)
*Custom molecule panel with file upload and placement configuration*

### Overview

The Custom Molecule tab allows you to insert user-provided molecules into liquid water regions. This enables simulation of arbitrary solutes, drugs, or other molecules in ice-water systems.

### Prerequisites

- Interface structure generated in Tab 2 (Interface Construction)
- Valid .gro and .itp files for your custom molecule
- Force field parameters for your molecule

### Workflow

1. **Generate Interface First**: Create an ice-water interface in Tab 2
2. **Upload GRO File**: Click "Upload GRO" to select coordinate file
   - Must contain atom coordinates in nm
   - Residue name in columns 6-10 (fixed-width GRO format)
3. **Upload ITP File**: Click "Upload ITP" to select topology file
   - Must include [ atomtypes ], [ moleculetype ], [ atoms ] sections
   - User-provided [ atomtypes ] required (no force field database)
4. **Resolve Validation Issues**: If residue name mismatch detected:
   - Option 1: Use residue name from GRO file
   - Option 2: Use molecule name from ITP file
5. **Choose Placement Mode**:
   - **Random**: Automatic placement with overlap checking
     - Set number of molecules to insert
     - Molecules placed randomly in liquid region
     - All-atom overlap checking prevents clashes
   - **Custom**: User-specified positions and rotations
     - Set center-of-mass position (X, Y, Z in nm)
     - Set rotation angles (alpha, beta, gamma in degrees, ZXZ Euler convention)
     - No overlap checking (user responsibility)
6. **Generate**: Click "Generate" button
7. **Visualize**: View custom molecules in distinct colors (purple, cyan, yellow)
8. **Export**: Use Ctrl+M or File menu to export for GROMACS

### GRO File Format

```
Custom Molecule
    8
    1DRUG    C1    1   1.000   2.000   3.000
    1DRUG    C2    2   1.500   2.500   3.500
    ...
    8
   5.000   5.000   5.000
```

- Line 1: Title (arbitrary)
- Line 2: Atom count
- Lines 3+: Atom records (residue name cols 6-10, atom name cols 11-15, coords cols 21-45)
- Last 3 lines: Box dimensions

### ITP File Requirements

```
[ atomtypes ]
; name  bond_type  mass  charge  ptype  sigma  epsilon
  C1        1      12.01  0.00    A     0.34   0.36
  ...

[ moleculetype ]
; name  nrexcl
  DRUG    3

[ atoms ]
; nr  type  resnr  residue  atom  cgnr  charge  mass
   1   C1    1     DRUG     C1    1    -0.15   12.01
   ...
```

Required sections:
- [ atomtypes ]: Force field parameters for each atom type
- [ moleculetype ]: Molecule definition
- [ atoms ]: Atom list with types and charges

### Validation

On file upload, QuickIce validates:
- Atom count consistency (GRO vs ITP)
- Residue name presence (GRO columns 6-10)
- Required ITP sections present

### Visualization

Custom molecules rendered with distinct colors:
- First custom molecule type: Purple
- Second custom molecule type: Cyan
- Third custom molecule type: Yellow

### GROMACS Export

Exported files include:
- .gro file with ice + water + custom molecules
- .top file with molecule ordering (SOL → custom molecules)
- tip4p-ice.itp (water topology)
- Custom .itp file (your uploaded topology, copied to output directory)

### Tips

- Prepare .itp files using force field tools (e.g., ACPYPE, CHARMM-GUI)
- Test molecule in vacuum simulation before inserting into interface
- Use Custom placement mode for precise positioning near interfaces
- Check overlap visually after Random placement
```
