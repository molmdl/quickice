**2. Update Workflow section with correct tab numbering:**
```
Tab 0 — Ice Generation:
1. Enter temperature, pressure, and molecule count
2. Click on phase diagram OR type values directly
3. Press Enter or click Generate button
4. View ranked candidates in dual 3D viewer
5. Use File menu to export PDB, GROMACS files, diagram, or screenshots

Tab 1 — Hydrate Config:
6. Switch to Hydrate Config tab
7. Select lattice type (sI, sII, sH) and guest type (empty, CH₄, etc.)
8. Set unit cell repetitions and generate hydrate structure
9. Export hydrate for GROMACS (Ctrl+H)

Tab 2 — Interface Construction:
10. Switch to Interface Construction tab
11. Select mode: Slab (layered), Pocket (water cavity), or Piece (ice in water)
12. Set box dimensions and mode-specific parameters
13. Select a candidate and click Generate Interface
14. View result (ice=cyan, water=cornflower blue)
15. Export interface for GROMACS (Ctrl+I)

Tab 3 — Custom Molecule:
16. Switch to Custom Molecule tab (requires interface from Tab 2)
17. Upload .gro and .itp files for your custom molecule
18. Choose placement mode (Random or Custom)
19. Set position/rotation (Custom mode) or molecule count (Random mode)
20. Click Generate to insert custom molecules
21. Export custom molecules for GROMACS (Ctrl+M)

Tab 4 — Solute Insertion:
22. Switch to Solute Insertion tab (requires interface from Tab 2)
23. Set concentration (mol/L) and select solute type (THF or CH₄)
24. Click Insert Solutes to place molecules in liquid region
25. Export solutes for GROMACS (Ctrl+L)

Tab 5 — Ion Insertion:
26. Switch to Ion Insertion tab (requires interface from Tab 2)
27. Set ion concentration and insert ions into interface
28. Export ions for GROMACS (Ctrl+J)
```
