8. Return the vtkMolecule

Do NOT create sphere/cylinder actors manually - vtkMoleculeMapper handles this automatically.

  <files>quickice/gui/vtk_utils.py</files>
  <action>
Add a `detect_hydrogen_bonds()` function that identifies H-bonds from molecular geometry.

Implementation details:
1. Create function `detect_hydrogen_bonds(candidate: Candidate, max_distance: float = 0.25) -> list[tuple[tuple[float, float, float], tuple[float, float, float]]]`
   - max_distance in nm (0.25 nm = 2.5 Å, typical H-bond threshold)
   - Returns list of (H_position, O_position) tuples for each H-bond
2. Algorithm:
   - Find all O and H atom positions (O atoms are at indices 0, 3, 6, ... i.e., every 3rd atom starting at 0)
   - H atoms are at indices 1, 2, 4, 5, 7, 8, ...
   - For each H atom, find O atoms that:
     a. Are NOT the O atom in the same water molecule (covalently bonded)
     b. Have H...O distance < max_distance
   - Return the H-O pairs as coordinates
3. Use numpy for distance calculations: `np.linalg.norm(h_pos - o_pos)`

Reference: H-bonds occur when H...O distance < ~2.5 Å (0.25 nm) and typically involve O-H...O geometry.
  </action>
