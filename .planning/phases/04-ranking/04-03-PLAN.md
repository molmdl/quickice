def normalize_scores(scores: list[float]) -> np.ndarray:
    """Normalize scores to 0-1 range using min-max scaling.
    
    Lower raw scores become lower normalized scores (0 = min, 1 = max).
    If all scores equal, returns zeros.
    """
```

Implementation:
1. Convert to numpy array
2. Get min and max
3. If max == min, return zeros (all same)
4. Return `(scores - min) / (max - min)`

**Key insight:** For energy and density scores, lower raw = better, so after normalization lower = better (closer to 0). For diversity, higher raw = better, so after normalization higher = better (closer to 1). The combined score formula will handle this correctly.

def rank_candidates(
    candidates: list[Candidate],
    weights: dict[str, float] | None = None
) -> RankingResult:
    """Rank candidates by combined scoring.
    
    Args:
        candidates: List of Candidate objects from Phase 3
        weights: Optional weight config (default: equal weights 1:1:1)
    
    Returns:
        RankingResult with sorted candidates and score breakdown
    """
```

Implementation:
1. Default weights: `{'energy': 1.0, 'density': 1.0, 'diversity': 1.0}`
2. Calculate raw scores for each component:
   - `energy_scores = [energy_score(c) for c in candidates]`
   - `density_scores = [density_score(c) for c in candidates]`
   - `diversity_scores = [diversity_score(c, candidates) for c in candidates]`
3. Normalize each component to 0-1
4. Calculate combined score for each candidate:
   - For energy/density: lower normalized = better, so use directly
   - For diversity: higher normalized = better, so invert: `1 - normalized_diversity`
   - Combined = `w_energy * norm_energy + w_density * norm_density + w_diversity * (1 - norm_diversity)`
   - Lower combined = better
5. Create RankedCandidate objects with all scores
6. Sort by combined_score (ascending, lower = better)
7. Assign ranks (1 = best)
8. Return RankingResult with metadata

**Important:** For diversity, invert the normalized score because:
- Raw diversity: higher = better
- Normalized: 1 = most diverse
- We want lower combined = better, so use `(1 - norm_diversity)` in combination

Include scoring_metadata with n_candidates and ideal_oo_distance.
