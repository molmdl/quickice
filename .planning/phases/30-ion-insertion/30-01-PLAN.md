Create quickice/structure_generation/ion_inserter.py with IonInserter class:

1. **Constants:**
   - AVOGADRO = 6.02214076e23  # mol^-1
   - NA_VDW_RADIUS = 0.190  # nm (190 pm)
   - CL_VDW_RADIUS = 0.181  # nm (181 pm)
   - NA_COLOR = (1.0, 0.84, 0.0)  # Gold
   - CL_COLOR = (0.56, 0.99, 0.56)  # Lime green
   - MIN_SEPARATION = 0.3  # nm minimum ion-ion distance

2. **calculate_ion_pairs(concentration_molar, liquid_volume_nm3) -> int:**
   - N_ions = C_M × V_L × NA
   - V_L = V_nm3 × 1e-24 (nm³ to L)
   - Return ion_pairs = N_ions // 2 (round to nearest int)

3. **replace_water_with_ions(InterfaceStructure, ion_pairs, config) -> IonStructure:**
   - Extract water molecules from molecule_index (mol_type == 'water')
   - Randomly select ion_pairs water molecules to replace
   - Replace each water with either Na+ or Cl- (alternating for neutrality)
   - Use cKDTree to check minimum distance to existing atoms
   - Update molecule_index with new ion entries

4. **Key logic:**
   - Only replace water in liquid region (skip ice molecules)
   - Ensure equal Na+ and Cl- counts (charge neutrality)
   - Check overlap using cKDTree from overlap_resolver

Reference: InterfaceStructure has `molecule_index: list[MoleculeIndex]` where each entry tracks (start_idx, count, mol_type).
