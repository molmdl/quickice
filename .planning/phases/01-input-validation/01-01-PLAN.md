<task type="auto">
  <name>Create setup.sh to activate env and set PYTHONPATH</name>
  <files>setup.sh</files>
  <action>
Create setup.sh at the project root:

```bash
#!/bin/bash
# QuickIce environment setup
# Usage: source setup.sh

# Activate conda environment
conda activate quickice

# Add project to PYTHONPATH so 'quickice' package is importable
export PYTHONPATH="${PYTHONPATH}:$(pwd)"

echo "QuickIce environment activated. Run 'quickice --help' for usage."
```

Make it executable but note: must be sourced, not executed directly.
  </action>
  <verify>cat setup.sh | grep -E "conda activate|PYTHONPATH"</verify>
  <done>setup.sh exists with conda activate and PYTHONPATH export</done>
</task>
