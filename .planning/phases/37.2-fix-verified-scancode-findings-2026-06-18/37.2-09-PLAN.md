Replace the triple-nested Python loop in `wrap_molecules_into_box` (lines 93-138) with vectorized numpy operations.

The outer molecule loop must remain (molecules must be processed independently for center computation), but the inner two loops (per-atom dimension loop and per-atom unwrap check) can be vectorized.

Replace lines 93-138 with:

```python
    for mol in molecule_index:
        start = mol.start_idx
        count = mol.count
        
        # Get positions for this molecule
        mol_positions = wrapped[start:start + count].copy()
        
        # Step 1: Unwrap atoms that are split across PBC (vectorized)
        # Use first atom as reference
        ref_pos = mol_positions[0]
        
        # Compute delta from reference for all atoms at once
        # delta shape: (count, 3)
        delta = mol_positions[1:] - ref_pos
        
        # Get box sizes for all 3 dimensions
        box_sizes = np.array([cell[d, d] for d in range(3)])
        
        # Compute unwrapping shifts: where delta > box/2, shift back; where delta < -box/2, shift forward
        # Vectorized across all atoms and dimensions
        shifts = np.zeros_like(delta)
        half_box = box_sizes / 2
        
        # Atoms "ahead" (delta > box/2) → shift back
        shifts[delta > half_box] -= box_sizes[np.where(delta > half_box)[1]]
        
        # Atoms "behind" (delta < -box/2) → shift forward
        shifts[delta < -half_box] += box_sizes[np.where(delta < -half_box)[1]]
        
        # Apply shifts to unwrap all atoms at once
        mol_positions[1:] += shifts
        
        # Step 2: Wrap the whole molecule into [0, box_size) using center
        center = np.mean(mol_positions, axis=0)
        
        # Vectorized center wrapping
        center_wrapped = np.mod(center, box_sizes)
        shift = center_wrapped - center
        
        # Apply shift to all atoms in molecule
        mol_positions += shift
        
        # Store wrapped positions
        wrapped[start:start + count] = mol_positions
    
    return wrapped
```

Key changes:
- `for i in range(1, len(mol_positions)):` → vectorized `delta = mol_positions[1:] - ref_pos`
- `for dim in range(3):` → vectorized across all 3 dims using `box_sizes` array
- Per-atom `if delta[dim] > box_size/2:` → numpy boolean indexing with `shifts[delta > half_box]`
- `for dim in range(3):` in center wrapping → `np.mod(center, box_sizes)` and `shift = center_wrapped - center`

IMPORTANT: The `np.where(delta > half_box)[1]` extracts the column (dimension) index to select the correct box_size for each violation. This is needed because shifts must use the dimension-specific box size, not a single value.
