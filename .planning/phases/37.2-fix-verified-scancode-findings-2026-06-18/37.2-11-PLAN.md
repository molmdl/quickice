**1. Add I/O-bound comment to per-atom GRO string formatting (#27 PERF-06):**

In gromacs_writer.py, find the per-atom loop that builds GRO format strings (the `lines.append(...)` calls in any GRO writer function). Add a comment at the top of the atom-writing loop in `write_ice_gro_file` (the first/most prominent GRO writer) explaining why the per-atom loop is acceptable:

Find the loop that iterates over atoms to build GRO lines. It typically looks like:
```python
    for i in range(n_atoms):
        ...
        lines.append(f"{res_num:5d}{res_name:5s}{atom_name:5s}...")
```

Add this comment BEFORE the loop:
```python
    # PERF-06 NOTE: This per-atom loop formats and appends GRO coordinate strings.
    # While this is O(N) in Python, the loop is I/O-bound (the writelines() call
    # that follows dominates execution time). The heterogeneous formatting
    # (different residue types, atom name column widths, and position formatting)
    # makes vectorization complex with minimal performance gain since the actual
    # bottleneck is disk I/O, not string formatting. Kept as explicit loop for
    # clarity and maintainability.
```

Add this comment in ONE location only: the `write_ice_gro_file` function's atom-writing loop, as it serves as the reference implementation. Other GRO writers follow the same pattern.

**2. Add WATER_VOLUME_NM3 citation and derivation (#32 R-07):**

In `quickice/structure_generation/types.py`, replace the current comment at line 24-25:

```python
# Volume per TIP4P-ICE water molecule in nm³ (at ~1 bar, 300K)
WATER_VOLUME_NM3: float = 0.0299
```

With:

```python
# Volume per TIP4P-ICE water molecule in nm³ (liquid phase, ~1 bar, 300K)
#
# Derivation: TIP4P-ICE liquid water at ~300K/1bar has density ≈ 0.997 g/cm³.
# V_molecule = M / (ρ × N_A) = 18.015 g/mol / (0.997 g/cm³ × 6.022e23 /mol)
#           = 30.0 Å³ = 0.0300 nm³
# The value 0.0299 nm³ (= 1/33.4 molecules/nm³) is consistent with the
# TIP4P-ICE model's liquid-phase density at ambient conditions.
#
# Reference: Abascal, J. L. F., Sanz, E., García Fernández, R., & Vega, C.
# (2005). A potential model for the phase diagram of TIP4P water.
# J. Chem. Phys. 122, 234511. DOI: 10.1063/1.1931662
#
# For TIP4P-ICE specifically: Abascal, J. L. F., & Vega, C. (2005).
# A general purpose model for the condensed phases of water: TIP4P/2005.
# J. Chem. Phys. 123, 234505. DOI: 10.1063/1.2121600
WATER_VOLUME_NM3: float = 0.0299
```

Note: The value 0.0299 is for LIQUID water volume estimation (used by inserters to compute how many solutes to insert in the liquid phase), NOT for ice volume. The derivation uses liquid water density at ~300K.
