Replace the `collect_all(pkg)` pattern in `quickice-gui.spec` with targeted `collect_submodules` + `collect_data_files` for each package. This prevents test data, documentation, and other non-runtime files from being bundled.

Replace the entire top section (lines 1-16):

```python
# -*- mode: python ; coding: utf-8 -*-
from PyInstaller.utils.hooks import collect_all

# Collect all data files, binaries, and hidden imports from packages
datas = [('quickice/data', 'quickice/data')]
binaries = []
hiddenimports = []

for pkg in ['iapws', 'genice2', 'genice_core', 'matplotlib', 'scipy', 'numpy', 'shapely', 'networkx', 'spglib']:
    try:
        tmp_ret = collect_all(pkg)
        datas += tmp_ret[0]
        binaries += tmp_ret[1]
        hiddenimports += tmp_ret[2]
    except Exception as e:
        print(f"Warning: Could not collect {pkg}: {e}")
```

With:

```python
# -*- mode: python ; coding: utf-8 -*-
from PyInstaller.utils.hooks import collect_submodules, collect_data_files, collect_binaries

# Targeted collection: collect submodules and data files separately,
# excluding test data, documentation, and other non-runtime files.
# This significantly reduces bundle size compared to collect_all().
datas = [('quickice/data', 'quickice/data')]
binaries = []
hiddenimports = []

for pkg in ['iapws', 'genice2', 'genice_core', 'matplotlib', 'scipy', 'numpy', 'shapely', 'networkx', 'spglib']:
    try:
        # Submodules: all Python import paths (hidden imports)
        hiddenimports += collect_submodules(pkg)
        # Data files: non-Python resources (config files, ITP files, etc.)
        datas += collect_data_files(pkg)
        # Binaries: compiled extensions (.so, .pyd files)
        binaries += collect_binaries(pkg)
    except Exception as e:
        print(f"Warning: Could not collect {pkg}: {e}")
```

IMPORTANT: `collect_data_files` only collects data files (not tests), `collect_submodules` only collects Python import paths, and `collect_binaries` only collects compiled extensions. The combination of these three replaces `collect_all` which bundles everything including test directories.
