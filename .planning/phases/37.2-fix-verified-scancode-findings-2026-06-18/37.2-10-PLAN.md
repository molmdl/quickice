**1. Path traversal check for CLI file inputs (#25 SEC-04):**

After the file existence and extension checks at lines 364-381, add path traversal validation. Insert after the extension check block (after line 381):

```python
            # SEC-04: Reject directory traversal outside working directory
            resolved_gro = gro_path.resolve()
            resolved_itp = itp_path.resolve()
            cwd = Path.cwd().resolve()
            if not resolved_gro.is_relative_to(cwd):
                report_progress(
                    f"Error: --custom-gro path resolves outside working directory: "
                    f"{resolved_gro} is not under {cwd}"
                )
                return 1
            if not resolved_itp.is_relative_to(cwd):
                report_progress(
                    f"Error: --custom-itp path resolves outside working directory: "
                    f"{resolved_itp} is not under {cwd}"
                )
                return 1
```

Also add the same check for `--custom-positions-file` in the `_parse_positions_csv` method (around line 161). After the `if not path.exists()` check, add:

```python
        # SEC-04: Reject directory traversal outside working directory
```
