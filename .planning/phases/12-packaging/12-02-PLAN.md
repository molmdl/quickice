<task type="auto">
  <name>Build Linux executable with PyInstaller</name>
  <files>dist/quickice-gui, dist/_internal/</files>
  <action>
    Build standalone executable using PyInstaller in onedir mode.
    
    **Build command:**
    ```bash
    pyinstaller --name quickice-gui \
                --onedir \
                --noconsole \
                --clean \
                quickice/gui/__main__.py
    ```
    
    **Explanation of flags:**
    - `--name quickice-gui`: Output executable name (matches CONTEXT.md)
    - `--onedir`: Create directory with executable + _internal/ for libraries (faster than --onefile)
    - `--noconsole`: Hide terminal window (GUI application, no CLI needed)
    - `--clean`: Clean cache before building
    - `quickice/gui/__main__.py`: Entry point module
    
    **Build process:**
    1. PyInstaller analyzes imports starting from `quickice/gui/__main__.py`
    2. Discovers all dependencies: PySide6, VTK, NumPy, SciPy, Matplotlib, GenIce2, etc.
    3. Creates `dist/quickice-gui` executable
    4. Creates `dist/_internal/` directory with all bundled libraries
    5. Build takes 5-15 minutes depending on system
    
    **Do NOT use spec file optimization:**
    Per CONTEXT.md: "Use PyInstaller defaults, add explicit excludes only if needed for build errors"
    
    After build completes, verify:
    ```bash
    ls -lh dist/
    ```
    
    Expected:
    - `dist/quickice-gui` executable (~10-50MB)
    - `dist/_internal/` directory with libraries (~500MB-2GB)
  </action>
  <verify>
    test -x dist/quickice-gui && test -d dist/_internal && echo "Executable built successfully"
  </verify>
  <done>
    dist/quickice-gui executable exists and is executable, dist/_internal/ contains bundled libraries
  </done>
</task>

<task type="auto">
  <name>Assemble distribution tarball</name>
  <files>quickice-v2.0.0-linux-x86_64.tar.gz</files>
  <action>
    Create distribution directory and package into tarball.
    
    **Create distribution structure:**
    ```bash
    # Create distribution directory
    mkdir -p dist/package
    
    # Copy executable
    cp -r dist/quickice-gui dist/package/
    
    # Copy bundled libraries
    cp -r dist/_internal dist/package/
    
    # Copy essential documentation
    cp README.md dist/package/
    cp LICENSE dist/package/
    
    # Copy GUI guide
    mkdir -p dist/package/docs
    cp docs/gui-guide.md dist/package/docs/
    
    # Copy license files
    cp -r licenses dist/package/
    
    # Create tarball
    cd dist
    tar -czf quickice-v2.0.0-linux-x86_64.tar.gz package/
    mv quickice-v2.0.0-linux-x86_64.tar.gz ../
    cd ..
    
    # Clean up package directory
    rm -rf dist/package
    ```
    
    **Distribution contents:**
    ```
    quickice-v2.0.0-linux-x86_64.tar.gz
    ├── quickice-gui           # Standalone executable
    ├── _internal/             # Bundled libraries
    ├── README.md              # Project documentation
    ├── LICENSE                # MIT license
    ├── docs/
    │   └── gui-guide.md       # GUI usage guide
    └── licenses/              # Dependency licenses
        ├── LGPL-3.0.txt       # PySide6/Qt
        ├── BSD-3-Clause.txt   # VTK, NumPy, SciPy
        ├── PSF-2.0.txt        # Matplotlib
        └── MIT.txt            # GenIce2
    ```
    
    **Verify tarball:**
    ```bash
    tar -tzf quickice-v2.0.0-linux-x86_64.tar.gz | head -20
    ```
    
    Should show the directory structure above.
  </action>

  </verify>
  <done>

  </done>
