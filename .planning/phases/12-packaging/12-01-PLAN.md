<task type="auto">
  <name>Collect bundled dependency license files</name>
  <files>licenses/LGPL-3.0.txt, licenses/BSD-3-Clause.txt, licenses/PSF-2.0.txt, licenses/MIT.txt</files>
  <action>
    Create `licenses/` directory and collect license files for all bundled dependencies.
    
    **Dependencies requiring licenses:**
    1. **PySide6/Qt** - LGPL-3.0 (CRITICAL - Qt requires LGPL-3.0.txt in distribution)
       - Download from: https://www.gnu.org/licenses/lgpl-3.0.txt
       - Save as: licenses/LGPL-3.0.txt
    
    2. **VTK, NumPy, SciPy** - BSD-3-Clause (all use same BSD-3-Clause license)
       - Get text from: https://opensource.org/licenses/BSD-3-Clause
       - Save as: licenses/BSD-3-Clause.txt
       - Single file covers all BSD-3-Clause dependencies
    
    3. **Matplotlib** - PSF-based license (matplotlib uses custom license based on PSF)
       - Extract license text from matplotlib package or use PSF-2.0
       - Save as: licenses/PSF-2.0.txt
       - Alternative: Download from https://matplotlib.org/stable/users/license.html
    
    4. **GenIce2** - MIT license (same as QuickIce)
       - Copy from existing LICENSE file or standard MIT text
       - Save as: licenses/MIT.txt
       - Note: GenIce2 author(s) may have different copyright holders
    
    **Download commands:**
    ```bash
    mkdir -p licenses
    
    # LGPL-3.0 for PySide6/Qt
    curl -o licenses/LGPL-3.0.txt https://www.gnu.org/licenses/lgpl-3.0.txt
    
    # BSD-3-Clause for VTK, NumPy, SciPy
    curl -o licenses/BSD-3-Clause.txt https://opensource.org/licenses/BSD-3-Clause
    
    # PSF-2.0 for Matplotlib (matplotlib uses PSF-style)
    curl -o licenses/PSF-2.0.txt https://opensource.org/licenses/Python-2.0
    
    # MIT for GenIce2 and project
    cp LICENSE licenses/MIT.txt
    ```
    
    After downloading, verify each file contains valid license text (not error pages).
  </action>
  <verify>
    test -f licenses/LGPL-3.0.txt && test -f licenses/BSD-3-Clause.txt && test -f licenses/PSF-2.0.txt && test -f licenses/MIT.txt && echo "All license files present"
  </verify>
  <done>
    licenses/ directory contains LGPL-3.0.txt, BSD-3-Clause.txt, PSF-2.0.txt, and MIT.txt with valid license text
  </done>
</task>
