## Types (types.py)

Create dataclasses for structured data:

```python
from dataclasses import dataclass, field
from typing import Optional
import numpy as np

@dataclass
class Candidate:
    """A single generated ice structure candidate."""
    positions: np.ndarray          # (N_atoms, 3) coordinates in nm
    atom_names: list[str]         # ["O", "H", "H", "O", "H", "H", ...]
    cell: np.ndarray              # (3, 3) cell vectors in nm
    nmolecules: int               # Actual number of water molecules
    phase_id: str                 # e.g., "ice_ih"
    seed: int                     # Random seed used for generation
    metadata: dict = field(default_factory=dict)  # density, T, P from Phase 2

@dataclass
class GenerationResult:
    """Result of generating multiple candidates."""
    candidates: list[Candidate]
    requested_nmolecules: int
    actual_nmolecules: int
    phase_id: str
    phase_name: str
    density: float
    was_rounded: bool  # True if actual != requested
```

## Errors (errors.py)

```python
class StructureGenerationError(Exception):
    """Base error for structure generation."""
    pass

class UnsupportedPhaseError(StructureGenerationError):
    """Phase not supported by GenIce."""
    def __init__(self, message: str, phase_id: str):
        super().__init__(message)
        self.phase_id = phase_id
```
