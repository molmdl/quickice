**Step 3: Seed Rotation.random() in CustomMoleculeInserter**

Change line 605 from:
```python
rotation = Rotation.random()
```
To:
```python
rotation = Rotation.random(random_state=self.seed)
```

Note: `Rotation.random(random_state=...)` accepts an int seed (not a `random.Random` instance). Using the same seed for both position and rotation is acceptable — they use different RNG engines (Python `random` vs numpy), so the sequences won't be correlated.

**Step 4: Seed Rotation.random() in SoluteInserter**

In `quickice/structure_generation/solute_inserter.py`, line 234 in `_generate_random_rotation_matrix()`:
```python
rotation = Rotation.random()
```
Change to:
```python
rotation = Rotation.random(random_state=self.rng.randint(0, 2**31 - 1))
```

Using `self.rng.randint(0, 2**31 - 1)` generates a reproducible seed from the seeded `random.Random` instance, so the rotation is reproducible given the same constructor seed. This is better than using a fixed seed because it produces different rotations for each call while remaining reproducible.

IMPORTANT: Do NOT change `self.rng.uniform()` calls in SoluteInserter — those are already using the seeded RNG correctly. Only fix `Rotation.random()` which is currently unseeded.
