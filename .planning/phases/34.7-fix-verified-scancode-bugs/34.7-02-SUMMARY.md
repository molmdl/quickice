## Decisions Made
- **WATER_ATOMS_PER_MOLECULE = 4** — placed after MOLECULE_TYPE_INFO dict as the canonical source for water atom count. Value 4 matches TIP4P-ICE model used throughout codebase.
- **seed=None default** — CustomMoleculeInserter seed defaults to None, maintaining backward compatibility. Existing callers (custom_molecule_worker.py, custom_molecule_panel.py) don't pass seed and get unseeded behavior matching previous code.
- **Rotation.random(random_state=self.seed)** in CustomMoleculeInserter — uses the same seed for both position and rotation. Different RNG engines (Python random vs numpy) mean sequences aren't correlated.
- **self.rng.randint(0, 2**31 - 1)** in SoluteInserter — generates a reproducible but varying seed from the seeded Random instance, so each call produces different rotations while remaining reproducible given the same constructor seed.
