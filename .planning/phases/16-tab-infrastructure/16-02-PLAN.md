  <action>
Implement Refresh candidates button to manually sync Tab 2 dropdown with Tab 1 candidates:

1. In MainWindow._setup_connections():
   - Connect InterfacePanel.refresh_requested signal:
     ```python
     self.interface_panel.refresh_requested.connect(self._on_refresh_candidates)
     ```

2. Add MainWindow._on_refresh_candidates() slot:
   ```python
   @Slot()
   def _on_refresh_candidates(self):
       """Refresh candidates in Tab 2 from Tab 1's current candidates."""
       result = self._viewmodel.get_last_ranking_result()
       if result and hasattr(result, 'ranked_candidates'):
           self.interface_panel.update_candidates(result.ranked_candidates)
   ```

3. In InterfacePanel:
   - Connect refresh button clicked to refresh_requested signal:
     ```python
     self.refresh_btn.clicked.connect(lambda: self.refresh_requested.emit())
     ```

5. Preserve selection when possible:
   - If selected candidate still exists after refresh, keep selection
   - If selected candidate removed, reset to first candidate
