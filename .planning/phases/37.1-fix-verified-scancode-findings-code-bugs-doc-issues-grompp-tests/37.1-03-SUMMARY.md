**DOC-C3:** Fixed exit codes table in `docs/cli-reference.md`. Removed phantom exit code 3 (structure generation error) which does not exist in any code path. Correct table now shows:
- 0 = Success
- 1 = Runtime error (phase mapping, structure generation, or general exception)
- 2 = Argument error (invalid or missing required flags)

**DOC-C4:** Added note about --gromacs no-op behavior in pipeline mode. When pipeline flags (--hydrate, --interface, --custom-gro, --solute-type, --ion-concentration) are present, GROMACS files are always generated regardless of --gromacs flag.

### Task 2: Add 18 missing pipeline flag sections (DOC-C2)

Added 4 new documentation sections with 18 flag subsections to `docs/cli-reference.md`:

**Hydrate Generation Flags (6 flags):**
- `--hydrate` — Enable hydrate generation mode
- `--lattice-type` — Lattice type (sI, sII, sH)
- `--guest` — Guest molecule (CH4, THF)
- `--supercell-x/y/z` — Supercell repetition counts
- `--cage-occupancy-small` — Small cage occupancy %
- `--cage-occupancy-large` — Large cage occupancy %

**Custom Molecule Insertion Flags (6 flags):**
- `--custom-gro` — Path to custom molecule .gro file
- `--custom-itp` — Path to custom molecule .itp file
- `--custom-placement` — Placement mode (random, custom)
- `--custom-count` — Number of molecules (random mode)
- `--custom-concentration` — Concentration in mol/L (random mode)
- `--custom-positions-file` — CSV file with positions (custom mode)

**Solute Insertion Flags (3 flags):**
- `--solute-type` — Solute type (CH4, THF)
- `--solute-concentration` — Solute concentration in mol/L
- `--solute-source` — Source (interface, custom)

**Ion Insertion Flags (2 flags):**
- `--ion-concentration` — Ion concentration in mol/L
- `--ion-source` — Source (interface, custom, solute)

Each subsection includes: description, choices/defaults, dependency requirements, and example commands.

Also updated the Example Scripts section to cross-reference the new dedicated flag sections.
