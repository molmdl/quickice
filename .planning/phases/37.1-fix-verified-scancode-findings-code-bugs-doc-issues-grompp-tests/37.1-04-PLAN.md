DOC-G2 — Fix THF ring description in quickice/gui/solute_panel.py line 209:

Change: `"THF (tetrahydrofuran): 4-membered ring, commonly used in hydrate studies\n"`
To: `"THF (tetrahydrofuran): 5-membered ring (4C + 1O), commonly used in hydrate studies\n"`

THF (tetrahydrofuran, C₄H₈O) is a 5-membered ring consisting of 4 carbon atoms and 1 oxygen atom. "Tetrahydro" refers to the 4 hydrogen atoms added to the parent furan ring, NOT the ring size.

DOC-G3 — Fix overlap tooltip in quickice/gui/custom_molecule_panel.py line 256:

Change: `"  - No overlap checking (user responsibility)"`
To: `"  - Overlap checking with warning dialog (user can override)"`

The code at lines 859-887 implements `_check_overlap_with_existing_positions()` which checks center-to-center distance with 0.25 nm threshold and shows a warning dialog asking "Add anyway?" when overlap is detected. The tooltip should reflect this actual behavior.
