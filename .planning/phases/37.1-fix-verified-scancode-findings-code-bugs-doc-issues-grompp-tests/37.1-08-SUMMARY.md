### Task 2: V-05 — Unknown atom handling in hydrate_generator.py

**Previous behavior:** Unknown atom types (not OW, HW1, HW2, MW, Me, C, H, NA, CL,
CH4 pattern, or THF pattern) were silently skipped with no molecule_index entry and
no warning. This caused molecule_index gaps leading to incorrect atom counting in
downstream GROMACS export.

**Fix applied:**
1. Added `logger.warning()` with atom name, index, and suggestion about unsupported guest
2. Added `molecule_index.append(MoleculeIndex(i, 1, "unknown"))` to track the atom
3. Kept `i += 1` to advance past the unknown atom

**Verification:** `grep -n 'logger.warning\|Unknown atom' hydrate_generator.py` confirms
warning and molecule_index entry. Full test suite (932 tests) passes.
