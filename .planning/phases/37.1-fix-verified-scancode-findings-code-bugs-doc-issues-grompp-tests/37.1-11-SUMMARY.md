## Verification Results

### CP-03: Concentration Range Validation

- `--solute-concentration 10.0` → error: "concentration must be between 0.0 and 5.0 mol/L, got 10.0"
- `--ion-concentration -1.0` → error: "concentration must be between 0.0 and 5.0 mol/L, got -1.0"
- `--custom-concentration 0.5` → accepted
- `--custom-concentration 0.0` → accepted (boundary)
- `--custom-concentration 5.0` → accepted (boundary)

### CP-03: Occupancy Range Validation

- `--cage-occupancy-small 150.0` → error: "cage occupancy must be between 0.0 and 100.0%, got 150.0"
- `--cage-occupancy-small 50.0` → accepted
- `--cage-occupancy-small 0.0` → accepted (boundary)
- `--cage-occupancy-small 100.0` → accepted (boundary)

### SEC-02: File Extension Validation

- Extension check for `.gro` suffix on --custom-gro (case-insensitive via .suffix.lower())
- Extension check for `.itp` suffix on --custom-itp (case-insensitive via .suffix.lower())
- Parser accepts any string (validation deferred to pipeline step)
- Non-matching extensions return exit code 1 with clear error message
