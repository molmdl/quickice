    a.scripts,

    exclude_binaries=True,

EXPLANATION of changes:
- `console=True` on Windows: allocates a console window so CLI mode stdout/stderr are visible. On Linux/macOS this flag is ignored (console always works).
- `hide_console='hide-late'`: PyInstaller 5.x+ feature that auto-hides the console window on Windows when the GUI starts. For CLI-only invocations, the console stays visible. On Linux/macOS this parameter is ignored.
