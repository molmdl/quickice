Create the interactive phase diagram widget with the following components:

1. **PhaseDiagramCanvas class** (inherits from FigureCanvasQTAgg):
   - Initialize with Figure(figsize=(6, 5), dpi=100)
   - Create axes with set_xlim(50, 500) for temperature (K)
   - Use log scale for pressure: set_ylim(0.1, 100000) MPa (or 0.1 to 10000 bar linear if preferred)
   - Add title "Ice Phase Diagram", xlabel "Temperature (K)", ylabel "Pressure (bar)"
   - Emit Signal `coordinates_selected(float, float)` when clicked

2. **Phase rendering** (reuse from quickice/output/phase_diagram.py):
   - Build shapely Polygon objects from `_build_phase_polygon_from_curves()` for each phase
   - Store polygons in `self._phase_polygons` dict for fast lookup
   - Use PHASE_COLORS and PHASE_LABELS from phase_diagram.py
   - Draw filled polygons with alpha=0.6, edgecolor='black', linewidth=1.5
   - Add phase labels at polygon centroids using PHASE_LABELS
   - Draw melting curves as black lines on top of regions

3. **Event handlers**:
   - `mpl_connect('motion_notify_event', self._on_hover)` - show live T,P in info panel
   - `mpl_connect('button_press_event', self._on_click)` - place marker, detect phase
   - Check `event.inaxes` and `event.xdata is not None` before processing
   - Convert pressure from bar to MPa for phase lookup (1 bar = 0.1 MPa)

4. **Marker handling**:
   - Store marker artist in `self._marker`
   - `set_marker(temperature, pressure)` - creates circle marker (markersize=8, markerfacecolor='red', markeredgecolor='black')
   - Use zorder=10 to place marker on top
   - Call `self.draw_idle()` after marker changes

5. **PhaseDetector class** (internal helper):
   - `__init__`: Build shapely Polygons from `_build_phase_polygon_from_curves()` for all 12 phases
   - `detect_phase(temperature, pressure)`:
     - Create Point(temperature, pressure_mpa)
     - Check `polygon.contains(point)` for each phase
     - Check `polygon.touches(point)` for boundary detection
     - Return (phase_short_name, is_boundary) tuple
     - Return ("Multiple phases possible", True) if on boundary between 2+ phases
     - Return (None, False) if no match (liquid region)

6. **PhaseDiagramPanel class** (inherits from QWidget):
   - QVBoxLayout containing PhaseDiagramCanvas and QLabel for info display
   - Connect `canvas.coordinates_selected` signal to update info label
   - Set crosshair cursor on canvas: `self.canvas.setCursor(Qt.CrossCursor)`
   - Info label shows "Click on diagram to select T, P" initially
   - On click: Update info label with phase name and coordinates, or "Multiple phases possible"

7. **Imports**:
   ```python
   from PySide6.QtWidgets import QWidget, QVBoxLayout, QLabel
   from PySide6.QtCore import Signal, Qt
   from matplotlib.backends.backend_qtagg import FigureCanvasQTAgg
   from matplotlib.figure import Figure
   from shapely.geometry import Point, Polygon
   from quickice.output.phase_diagram import (
       _build_phase_polygon_from_curves, PHASE_COLORS, PHASE_LABELS
   )
   ```

8. **Key implementation notes**:
   - Log scale: If using log scale, matplotlib event.xdata/ydata handles conversion automatically
   - Pressure units: Canvas shows bar, lookup_phase expects MPa - convert with `pressure_mpa = pressure_bar * 0.1`
   - Do NOT recreate polygons on every hover/click - build once in `__init__`
   - Use `draw_idle()` not `draw()` for better performance
