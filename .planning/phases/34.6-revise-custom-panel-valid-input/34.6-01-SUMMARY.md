## Decisions Made

1. **Generic residue name suppression**: Defined a fixed set of generic residue names (MOL, UNK, LIG, XXX, RES) that are commonly used in computational chemistry as placeholders. 2. **Explicit button enable**: Added an explicit `validate_button.setEnabled(True)` call after successful validation to ensure the button remains enabled regardless of the preview dialog response. This makes the intent clear and ensures consistent button state.
