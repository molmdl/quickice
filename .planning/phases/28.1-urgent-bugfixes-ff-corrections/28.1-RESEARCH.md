## Bug Analysis

### Bug 1: Hydrate Generation Error

**Root Cause:** CO2/H2 removed from UI (hydrate_panel.py guest_combo) but code references remain in:
- `types.py` - GUEST_MOLECULES dict includes co2/h2 entries
- `hydrate_generator.py` - _build_genice_options() maps guest types to GenIce2 options
- `hydrate_export.py` - _get_guest_itp_path() searches for .itp files

**Code Evidence:**
```python
# quickice/structure_generation/types.py lines 92-106
GUEST_MOLECULES: dict[str, dict[str, Any]] = {
    "ch4": {...},
    "thf": {...},
    "co2": {...},  # Should be removed
    "h2": {...},   # Should be removed
}

# quickice/structure_generation/hydrate_generator.py lines 97-107
guest = config.guest_type
if guest == "ch4":
    guest_param = "me"
elif guest == "thf":
    guest_param = "thf"
elif guest == "co2":
    guest_param = "co2"  # No longer valid
elif guest == "h2":
    guest_param = "h2"   # No longer valid
```

**Fix Required:**
1. Remove co2/h2 from GUEST_MOLECULES in types.py (line 92-106)
2. Remove co2/h2 from guest_fits in HYDRATE_LATTICES in types.py
3. Remove co2/h2 parameter mapping in hydrate_generator.py (lines 102-106)
4. Remove co2/h2 entry point in hydrate_export.py _get_guest_itp_path()

### Bug 2: Ion Insertion Error

**Root Cause:** Non-existent method being called.

**Code Evidence in main_window.py lines 675-680:**
```python
inserter = IonInserter()
ion_structure = inserter.insert_ions(  # <-- METHOD DOES NOT EXIST!
    interface,
    config,
    liquid_volume
)
```

Looking at ion_inserter.py (lines 39-277):
- Class `IonInserter` has methods: `__init__`, `calculate_ion_pairs`, `replace_water_with_ions`
- But NO method called `insert_ions()`

However, there IS a module-level function `insert_ions()` defined at lines 246-277:
```python
def insert_ions(
    structure: IonStructure,
    concentration_molar: float,
    liquid_volume_nm3: float | None = None,
    seed: int | None = None,
) -> IonStructure:
```

**Fix Required:**
Change main_window.py line 675-680 from:
```python
inserter = IonInserter()
ion_structure = inserter.insert_ions(interface, config, liquid_volume)
```

To use the module-level function:
```python
liquid_volume = self.ion_panel.get_liquid_volume()
ion_structure = insert_ions(interface, config.concentration_molar, liquid_volume)
```

Or call the class methods directly:
```python
inserter = IonInserter(config)
ion_pairs = inserter.calculate_ion_pairs(config.concentration_molar, liquid_volume)
ion_structure = inserter.replace_water_with_ions(interface, ion_pairs)
```
