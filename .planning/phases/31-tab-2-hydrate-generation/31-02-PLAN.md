Create `quickice/gui/hydrate_renderer.py` with rendering functions:

1. **Imports**:
   - vtkMoleculeMapper, vtkMolecule, vtkActor from vtkmodules.all
   - HydrateStructure from types
   - numpy as np

2. **Water Framework Rendering (lines style)**:
   Create function `create_water_framework_actor(structure: HydrateStructure) -> vtkActor`:
   ```
   - Extract water molecule positions from molecule_index where mol_type == "water"
   - Build vtkMolecule with water atoms (OW, HW1, HW2, MW)
   - Create vtkMoleculeMapper with SetRenderAtoms(False), SetRenderBonds(True)
   - Configure for line/bond-only rendering
   - Return vtkActor with water mapper
   ```

3. **Guest Molecule Rendering (ball-and-stick)**:
   Create function `create_guest_actor(structure: HydrateStructure) -> vtkActor`:
   ```
   - Extract guest molecule positions (mol_type != "water")
   - Build vtkMolecule with guest atoms (C/H for CH4, C/O/H for THF)
   - Create vtkMoleculeMapper with SetRenderAtoms(True), SetRenderBonds(True)
   - UseBallAndStickSettings() for ball-and-stick style
   - Set CPK colors (C=gray 0.6,0.6,0.6, O=red 1,0,0, H=white 1,1,1)
   - Return vtkActor with guest mapper
   ```

4. **Combined Rendering Function**:
   Create function `render_hydrate_structure(structure: HydrateStructure) -> list[vtkActor]`:
   ```
   - Call create_water_framework_actor(structure) -> water_actor
   - Call create_guest_actor(structure) -> guest_actor
   - Return [water_actor, guest_actor]
   ```

5. **Atom-by-atom molecule builder helper**:
   Create `_build_vtk_molecule(positions, atom_names) -> vtkMolecule`:
   - Iterate through positions and atom_names
   - Add atoms to vtkMolecule with correct element detection
   - Connect bonded atoms (detect bonds based on distance)
   - Return configured vtkMolecule

Use CPK coloring for guests:
- Carbon (C): RGB (0.6, 0.6, 0.6) gray
- Oxygen (O): RGB (1.0, 0.0, 0.0) red
- Hydrogen (H): RGB (1.0, 1.0, 1.0) white
