**Background on Molecule Count:**
- User specifies a minimum number of molecules (e.g., 216)
- GenIce2 creates supercells to satisfy space group symmetry
- Actual count can be larger (e.g., 432 molecules = 2× supercell)
- This is documented in `gromacs_writer.py` but not clear in GUI
