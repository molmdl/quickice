<task type="auto">
  <name>Task 1: Create GROMACS file writer module</name>
  <files>quickice/output/gromacs_writer.py</files>
  <action>
Create quickice/output/gromacs_writer.py with two functions:

1. write_gro_file(candidate: Candidate, filepath: str) - Write GROMACS .gro format:
   - Title line: "Ice structure {phase_id} exported by QuickIce\n"
   - Number of atoms: nmol * 4 (4-point water: O, H1, H2, MW)
   - Atom lines in format: i5, a5, a5, i5, 3f8.3 (positions in nm)
   - Atom order per molecule: OW1, HW1, HW2, MW (massless virtual site)
   - Residue name: "SOL"
   - Box vectors: triclinic format v1(x) v2(y) v3(z) v1(y) v1(z) v2(x) v2(z) v3(x) v3(y)

2. write_top_file(candidate: Candidate, filepath: str) - Write GROMACS .top format:
   - [ defaults ] section with nbfunc=1, comb-rule=2, gen-pairs=yes
   - [ atomtypes ] section with OW_ice, HW_ice, MW types
   - [ moleculetype ] section: SOL with nrexcl=2
   - [ atoms ] section: 4 atoms (OW_ice, HW_ice, HW_ice, MW) with charges
   - [ settles ] section: TIP4P geometry (doh=0.09572, dhh=0.15139)
   - [ virtual_sites3 ] section: MW from OW, HW1, HW2 with a=b=0.13458335
   - [ exclusions ] section
   - [ system ] section with phase_id
   - [ molecules ] section with SOL count

Use exact column widths per GROMACS specification. Reference Candidate from quickice.structure_generation.types.
  </action>
  <verify>Import and call functions with dummy Candidate - verify no import errors</verify>
  <done>write_gro_file produces valid .gro with correct 4-point water format, write_top_file produces valid .top with moleculetype</done>
