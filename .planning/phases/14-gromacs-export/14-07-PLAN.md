## Gap Being Closed

**From user feedback:**
1. CRITICAL BUG: GROMACS export crashes with `AttributeError: 'InputPanel' object has no attribute 'temperature_spinbox'`
2. Unclear how to choose water model for GROMACS export (user confusion)

**Root Cause Analysis:**
- `main_window.py` lines 478-479 try to access `self.input_panel.temperature_spinbox.value()` and `self.input_panel.pressure_spinbox.value()`
- But `InputPanel` class (in `quickice/gui/view.py`) does NOT have `temperature_spinbox` or `pressure_spinbox` attributes
- InputPanel has `temp_input` and `pressure_input` (QLineEdit widgets)
- InputPanel provides `get_temperature()` and `get_pressure()` methods that return validated float values

**Water Model:**
- TIP4P-ICE water model is hardcoded in the GROMACS exporter (see `gromacs_writer.py`)
- No UI indicates which model is used
- Should add clarification to the export dialog/message
</context>
