  <how-to-verify>
1. Launch QuickIce GUI:
   ```
   python -m quickice.gui
   ```

2. **Test PDB Export:**
   - Click on the phase diagram to select a condition (e.g., click in Ice Ih region)
   - Click Generate button
   - Wait for generation to complete
   - Go to File → Save PDB (or Ctrl+S)
   - Verify file dialog opens with suggested filename (e.g., "ice_ih_250K_100bar_c1.pdb")
   - Choose location and save
   - Open the .pdb file in a text editor to verify it has CRYST1 and ATOM records
   - Try saving again to same location - verify overwrite prompt appears

3. **Test Diagram Export:**
   - Go to File → Save Diagram (or Ctrl+D)
   - Verify file dialog opens with "phase_diagram.png" default
   - Choose PNG or SVG format
   - Save and open the image file to verify diagram is visible with all phase labels

4. **Test Viewport Export:**
   - After generating a structure (3D viewer should be visible)
   - Go to File → Save Viewport (or Ctrl+Shift+S)
   - Verify file dialog opens with "ice_structure.png" default
   - Save and open the image to verify 3D structure is captured (not black)

5. **Test Phase Info Display:**
   - Click on different regions of the phase diagram
   - Verify log panel expands and shows:
     - Phase name (e.g., "Ice Ih")
     - Temperature and pressure
     - Density (e.g., "0.9167 g/cm³")
     - Structure type (e.g., "Hexagonal (P6₃/mmc)")
     - Citation reference
   - Click in vapor region and verify "unknown" phase is handled

6. **Test Help Tooltips:**
   - Hover over the ? icon next to Temperature label
   - Verify tooltip appears with temperature help text
   - Repeat for Pressure and Molecule count ? icons
   - Verify each tooltip shows relevant help text

7. **Test Error Cases:**
   - Try File → Save PDB before generating - verify warning dialog appears
   - Try File → Save Viewport before generating - verify warning dialog appears
