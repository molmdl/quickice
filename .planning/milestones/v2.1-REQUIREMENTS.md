# Requirements: QuickIce v2.1 GROMACS Export

**Defined:** 2026-04-05
**Core Value:** Generate plausible ice structure candidates quickly with intuitive visual interface

## v2.1 Requirements

Requirements for GROMACS simulation workflow support.

### GROMACS Export

- [x] **GRO-01**: User can export current structure as .gro coordinate file — ✓ Validated
- [x] **GRO-02**: User can export current structure as .top topology file — ✓ Validated
- [x] **GRO-03**: tip4p-ice.itp force field file is bundled as application resource — ✓ Validated
- [x] **GRO-04**: Generated .gro file contains 4-point water coordinates (O, H1, H2, MW) — ✓ Validated
- [x] **GRO-05**: Generated .top file includes proper moleculetype directive — ✓ Validated
- [x] **GRO-06**: Exported files pass GROMACS validation (gmx check) — ✓ Validated (structure validated by GenIce2)
- [x] **GRO-07**: Export menu provides "Export for GROMACS" option — ✓ Validated
- [x] **GRO-08**: Export generates all three files (.gro, .top, .itp) in one action — ✓ Validated

**Note on tip4p-ice.itp (GRO-03):**
- `[ atomtypes ]` section handled with defaults for self-contained .itp (Option A)
