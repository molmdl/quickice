## v4.0 Requirements

Molecule insertion milestone: hydrates + ions + custom molecules + enhanced 3D viewer

### Hydrates (Tab 2)

- [x] **HYDR-01**: User can select hydrate structure type (sI, sII, sH) ✓ (Phase 29)
- [x] **HYDR-02**: User can select built-in guest molecule (CH4, THF) ✓ (Phase 29)
- [x] **HYDR-03**: User can specify cage occupancy (which cages get which guests) ✓ (Phase 29)
- [x] **HYDR-04**: User can set supercell repetition (nx, ny, nz) ✓ (Phase 29)
- [x] **HYDR-05**: System generates hydrate structure via GenIce2 ✓ (Phase 29)
- [x] **HYDR-06**: 3D viewer displays guest molecules in distinct style from ice framework ✓ (Phase 31)
- [x] **HYDR-07**: GROMACS export produces valid .gro/.top/.itp with guest molecules ✓ (Phase 31)
- [x] **HYDR-08**: System displays hydrate unit cell info (cage counts, types) ✓ (Phase 31)

### Ions (Tab 4)

- [x] **ION-01**: User can specify NaCl concentration (mol/L or g/kg) ✓ (Phase 30)
- [x] **ION-02**: System auto-calculates ion count from concentration and volume ✓ (Phase 30)
- [x] **ION-03**: System replaces water molecules (not ice) with ions ✓ (Phase 30)
- [x] **ION-04**: System enforces charge neutrality (equal Na+ and Cl-) ✓ (Phase 30)
- [x] **ION-05**: Ion placement avoids overlap with existing atoms ✓ (Phase 30)
- [x] **ION-06**: 3D viewer renders ions as VDW spheres (Na+ yellow/gold, Cl- green) ✓ (Phase 30)
- [x] **ION-07**: GROMACS export includes bundled Na+/Cl- ion parameters ✓ (Phase 30)

### Custom Molecules (Tab 4)

- [ ] **CUST-01**: User can upload custom .gro file ⟳ Deferred to v4.5
- [ ] **CUST-02**: User can upload custom .itp topology file ⟳ Deferred to v4.5
- [ ] **CUST-03**: System validates .gro/.itp consistency (atom counts) ⟳ Deferred to v4.5
- [ ] **CUST-04**: User can select random placement mode in liquid phase ⟳ Deferred to v4.5
- [ ] **CUST-05**: System checks overlap after custom molecule placement ⟳ Deferred to v4.5
- [ ] **CUST-06**: 3D viewer renders custom molecules distinct from water/ions ⟳ Deferred to v4.5
- [ ] **CUST-07**: GROMACS export includes custom molecule .itp ⟳ Deferred to v4.5

### 3D Viewer Enhancements

- [ ] **VIEW-01**: System renders 5 molecule types (ice, liquid, ions, small molecules, large molecules) ⟳ Deferred to v4.5
- [ ] **VIEW-02**: User can toggle visibility per molecule type ⟳ Deferred to v4.5
- [ ] **VIEW-03**: User can select display style per molecule type (lines, ball-stick, VDW, stick) ⟳ Deferred to v4.5
- [ ] **VIEW-04**: User can select color per molecule type ⟳ Deferred to v4.5

### GROMACS Export (Cross-Cutting)

- [x] **GRO-01**: Export produces .top with multiple [moleculetype] sections ✓ (Phase 29)
- [x] **GRO-02**: Export includes correct [molecules] counts per type ✓ (Phase 29)
- [x] **GRO-03**: User-provided .itp files included or referenced in export ✓ (Phase 29)

### Water Density (v3.5 Carryover)

- [ ] **WATER-01**: Water density calculated from T/P via IAPWS-95 ⟳ Pending
- [x] **WATER-02**: Water density displayed in Tab 1 info panel ✓ (Phase 30)
- [ ] **WATER-03**: Interface liquid spacing matches target water density ⟳ Pending
- [ ] **WATER-04**: IAPWS density lookup cached (@lru_cache) ⟳ Pending
