# Milestone v2.5: Seawater Phase Diagram

**Status:** PLANNED
**Phases:** 15-16
**Total Requirements:** 13

## Overview

Add seawater/saltwater ice phase diagram support with salinity-temperature (S-T) visualization and salt ion structure generation.

## Phases

### Phase 15: Seawater S-T Diagram Widget

**Goal:** Users can view and interact with salinity-temperature phase diagram showing freezing point depression and phase regions

**Requirements:** SEA-01, SEA-02, SEA-03, SEA-04, SEA-05, SEA-06, UI-01, UI-02, UI-03

**Dependencies:** None (new diagram widget)

**Success Criteria (Observable User Behaviors):**

1. **User can view S-T phase diagram** — Diagram displays with salinity (0-12%) on X-axis and temperature on Y-axis, with freezing point depression curve visible

2. **User can click on diagram to select conditions** — Clicking on diagram places marker at selected salinity and temperature; coordinates displayed

3. **Selected values update input fields** — Clicking diagram or entering S,T values updates both diagram marker and input fields bidirectionally

4. **Diagram shows phase regions** — Visual distinction between "Sea Ice" region (below freezing curve) and "Liquid Seawater" region (above curve)

5. **Seawater is separate tab** — New tab in GUI labeled "Seawater" alongside existing "Pure Water" tab; switching preserves state

6. **Help explains S-T vs T-P** — Tooltip or help text explains that seawater has single phase (Ice Ih + brine) vs pure water's 12 polymorphs

---

### Phase 16: Salt Ion Structure Generation

**Goal:** Users can generate ice structures containing salt ions proportional to selected salinity

**Requirements:** SALT-01, SALT-02, SALT-03, SALT-04

**Dependencies:** Phase 15 (seawater diagram), Phase 14 (GROMACS export, for SALT-04)

**Success Criteria (Observable User Behaviors):**

1. **Generated structure includes Na+ and Cl- ions** — Output structure contains sodium and chloride ions in addition to water molecules

2. **Ion count proportional to salinity** — At 35 ppt salinity, ~2-3% of molecules are ions; count scales with selected S value

3. **Ions appear in PDB export** — Exported PDB files include ion atoms (NA, CL residue names) with correct coordinates

4. **Ions appear in GROMACS export** — If v2.1 complete, .top file includes ion moleculetypes and exported .gro contains ion coordinates

## Technical Notes

- IAPWS-08 standard provides freezing point calculation via `_Tf()` function
- Seawater ice is Ice Ih with brine pockets (not high-pressure polymorphs)
- Salinity range 0-12% covers all natural seawater
- Ion placement uses random distribution avoiding ice lattice sites
