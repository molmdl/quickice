# Milestone v3.0: Liquid-Solid Interface

**Status:** PLANNED
**Phases:** 17
**Total Requirements:** 11

## Overview

Generate ice-water interface structures combining crystalline ice layer with liquid water layer, with visual distinction and export capabilities.

## Phases

### Phase 17: Ice-Water Interface Generation

**Goal:** Users can generate and visualize combined ice-water interface structures for studying interfacial phenomena

**Requirements:** INT-01, INT-02, INT-03, INT-04, INT-05, INT-06, VIS-01, VIS-02, VIS-03, EXP-01, EXP-02

**Dependencies:** None (new generation capability)

**Success Criteria (Observable User Behaviors):**

1. **User can generate interface structure** — User selects "Generate Interface" option, receives structure combining ice and liquid layers

2. **Interface combines ice + liquid layers** — Generated structure contains distinct crystalline ice region (from GenIce2) and liquid water region (rule-based placement)

3. **Interface displays in VTK viewer** — 3D viewport shows combined structure; user can rotate, zoom, toggle representations

4. **Ice and liquid visually distinct** — Different colors or styling distinguish ice molecules from liquid molecules; legend or indicator shows mapping

5. **Hydrogen bonds across boundary** — H-bond visualization shows connections between ice and liquid regions at interface

6. **Interface orientation selectable** — User can choose basal or prism plane orientation for ice layer

7. **Layer thickness ratio configurable** — User can adjust ice:water ratio (e.g., 50:50, 70:30)

8. **Exports as single PDB file** — Interface structure exports to PDB with all atoms (ice + liquid + ions if seawater mode)

9. **GROMACS export available** — If v2.1 complete, interface exports as GROMACS files with ice and water moleculetypes

## Technical Notes

- GenIce2 generates ice layer; separate liquid generator creates water configuration
- Rule-based liquid placement uses randomized positions avoiding ice lattice
- Interface plane defined by crystallographic orientation
- VTK visualization extends existing viewer with phase coloring
