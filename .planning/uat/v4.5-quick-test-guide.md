# v4.5 Quick Test Guide

**QuickIce v4.5 Solute & Custom Molecule Insertion**
**Format:** Tab: Actions (What to Check)

---

## Workflow 1: Application Startup & Tab Structure

Launch app → Verify 6 tabs visible: Ice(0) Hydrate(1) Interface(2) Custom(3) Solute(4) Ion(5) → Click each tab → Verify no crashes.

---

## Workflow 2: Ice Generation

Tab 0 (Ice): Select "Ice Ih", click "Generate Ice" → Verify ice structure appears in 3D viewer.

---

## Workflow 3: Hydrate Generation

Tab 1 (Hydrate): Select "sI" structure, select "CH4" guest, click "Generate Hydrate" → Verify hydrate structure with guests in 3D viewer → Export (Ctrl+S) → Verify .top includes `ch4.itp` with `CH4_HYD` moleculetype.

---

## Workflow 4: Interface Generation

Tab 0 (Ice): Generate Ice Ih → Tab 2 (Interface): Select "slab" mode, click "Create Interface" → Verify interface structure shows ice + water phases in different colors.

---

## Workflow 5: Custom Molecule Upload & Insertion

Tab 2 (Interface): Create interface → Tab 3 (Custom): Click "Upload GRO" → Select `quickice/data/custom/etoh.gro` → Click "Upload ITP" → Select `quickice/data/custom/etoh.itp` → Verify "Validation successful" → Select "Random" mode → Set count to 5 → Click "Generate" → Verify 5 ethanol molecules in liquid region → Verify molecules rendered in purple/cyan ball-and-stick.

---

## Workflow 6: Solute Insertion

Tab 2 (Interface): Create interface → Tab 4 (Solute): Verify "Source: Interface" selected → Set concentration to 1.0 M → Select "CH4" → Click "Insert Solutes" → Verify solutes appear in liquid region only → Verify ball-and-stick rendering → Export (Ctrl+S) → Verify .top includes `ch4_liquid.itp` with `CH4_L` moleculetype.

---

## Workflow 7: Ion Insertion

Tab 2 (Interface): Create interface → Tab 5 (Ion): Verify "Source: Interface" selected → Set concentration to 0.15 M → Click "Insert Ions" → Verify Na+/Cl- ions appear in liquid region → Verify equal ion counts → Export (Ctrl+S) → Verify .gro and .top files contain ions.

---

## Workflow 8: Complete Workflow Chain (Interface → Custom → Solute → Ion)

Tab 2 (Interface): Create slab interface → Tab 3 (Custom): Upload etoh.gro/itp, Random mode, count 3, Generate → Tab 4 (Solute): Change Source to "Custom Molecule", concentration 0.5 M THF, Insert → Tab 5 (Ion): Change Source to "Solute", concentration 0.1 M, Insert → Export from Ion tab → Verify .gro contains ice + water + ethanol + THF + Na + Cl.

---

## Workflow 9: GROMACS Export Verification

Terminal: `ls quickice/data/ch4_liquid.itp quickice/data/thf_liquid.itp` → Verify both files exist → `grep moleculetype quickice/data/ch4_liquid.itp` → Verify shows `CH4_L` → `grep moleculetype quickice/data/thf_liquid.itp` → Verify shows `THF_L`.

---

## Workflow 10: Keyboard Shortcuts

Tab 0 (Ice): Generate ice → Press Ctrl+S → Verify export dialog appears for Ice tab → Tab 3 (Custom): Generate custom molecules → Press Ctrl+S → Verify export dialog appears for Custom tab.

---

## Workflow 11: Integration Tests (Automated)

Terminal: `pytest tests/test_integration_v35.py -v` → Verify all 11 tests pass.
